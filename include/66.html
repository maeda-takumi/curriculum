第4章：ポートフォリオの作り方〔35分〕
4-1. なぜポートフォリオが必要か
ポートフォリオとは、「自分の作品集」です。購入者は「この人に頼んだらどんなものができるか」を知りたいので、サンプルがないと購入してもらえません。

📂 ポートフォリオの役割
・「何ができるか」を視覚的に伝える
・スキルレベルを証明する
・購入者の「この人に頼みたい」を引き出す
・価格の妥当性を示す
4-2. ポートフォリオの作り方（実績がない場合）

🎨 実績がなくてもポートフォリオを作る方法
【方法1】架空の案件を想定して作る
→ 「もし〇〇さんから依頼されたら」と仮定
→ カフェのSNS画像、ハンドメイド作家のバナーなど
【方法2】自分のSNS用に作った画像を使う
→ 楽天ROOM、Instagram用に作った画像
→ 実際に使っているものは「実績」として使える
【方法3】友人・知人の依頼を無料or格安で受ける
→ 実績作りのために最初は無料でもOK
→ 「ポートフォリオに掲載させてください」と伝える
【方法4】テンプレートをアレンジして作る
→ Canvaのテンプレートを自分なりにカスタマイズ

6-2. ランサーズの登録と使い方

📝 ランサーズ登録手順
【Step1】ランサーズにアクセス
→ https://www.lancers.jp
→ 「無料で会員登録」をクリック
【Step2】メールアドレスで登録
→ メール認証→パスワード設定
【Step3】プロフィールを充実させる
→ 自己紹介、スキル、ポートフォリオを登録
→ 本人確認をするとランクUP（信頼度UP）
【案件の探し方】
→ 「仕事を探す」→カテゴリ「デザイン」→「バナー・画像」
→ 「初心者歓迎」で絞り込み
→ 気になる案件に提案（応募）
6-3. クラウドワークスの登録と使い方

📝 クラウドワークス登録手順
【Step1】クラウドワークスにアクセス
→ https://crowdworks.jp
→ 「無料で利用を開始」をクリック
【Step2】メールアドレスで登録
→ メール認証→基本情報入力
【Step3】プロフィールを設定
→ 自己紹介、スキル、実績を登録
→ 「クラウドワーカー」として登録
【案件の探し方】
→ 「仕事を探す」→「デザイン」→「画像・バナー」
→ 「未経験可」「初心者歓迎」で絞り込み
→ 応募フォームから提案

6-5. クラウドソーシングで稼ぐコツ

💡 クラウドソーシングで選ばれるコツ
【コツ1】素早く提案する
→ 案件公開後、早く提案した人が有利
→ 新着案件は毎日チェック
【コツ2】提案文をカスタマイズする
→ テンプレのコピペだけはNG
→ 案件内容に合わせて「この部分に貢献できます」と書く
【コツ3】最初は低単価でも受ける
→ 実績がないと選ばれにくい
→ 最初の5〜10件は実績作りと割り切る
【コツ4】評価を大切にする
→ 高評価が増えると、選ばれやすくなる
→ 丁寧な対応、納期厳守、期待以上の品質を心がける

7-2. 単価アップの方法

📈 単価を上げる5つの方法
【方法1】オプションを追加する
→ 「お急ぎ対応＋500円」「修正追加＋300円」
→ 基本価格は据え置きで、収益UP
【方法2】セット販売にする
→ 「1枚1,000円」→「5枚セット4,000円」
→ まとめ買いでお得感を出す
【方法3】プレミアム版を作る
→ 「スタンダード3,000円」「プレミアム5,000円」
→ プレミアムは修正無制限、納期短縮など
【方法4】専門特化する
→ 「なんでも作ります」より「美容サロン専門」の方が高単価
→ 特定ジャンルで実績を積む
【方法5】継続プランを作る
→ 「月額10,000円で月10枚まで」など
→ 安定収入につながる
7-3. リピーターを獲得する方法

🔁 リピーターを増やすコツ
【コツ1】期待以上の品質を出す
→ 「思ったより良かった！」と思ってもらう
→ 少しのプラスアルファ（サイズ違いも添える等）
【コツ2】対応を丁寧にする
→ 返信は早く、言葉遣いは丁寧に
→ 「この人に頼むと安心」と思ってもらう
【コツ3】納品時に次回の提案をする
→ 「また必要な際はお声がけください」
→ 「継続プランもご用意しています」
【コツ4】フォローアップする
→ 納品1〜2週間後に「その後いかがですか？」と連絡
→ 次の依頼につながることも
【コツ5】リピーター特典を作る
→ 「2回目以降10%OFF」「3回目は修正回数+1回」
7-4. 評価を集める重要性

⭐ 高評価を集めるために
【なぜ評価が重要か】
→ 購入者は評価を見て判断する
→ 評価が高いと検索上位に表示されやすい
→ 高評価が多いと単価を上げても売れる
【高評価を得るために】
→ 納期を守る（早いとなお良い）
→ 修正に嫌な顔をしない
→ こまめな連絡で安心感を与える
→ 完成度の高いものを納品する
【評価を書いてもらうコツ】
→ 納品時に「評価いただけると励みになります」と一言
→ 押し付けがましくならないように

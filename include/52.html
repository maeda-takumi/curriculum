3-1. 購入までの「心理フロー」を理解する
読者が有料記事を購入するまでには、心理的な「ステップ」があります。このステップを理解し、各ステップに対応する文章を配置するのが、セールスライティングの本質です。

🧠 購入の5ステップ心理フロー
【STEP1：気づく】「あ、これ自分のことだ」
→ 対応ライティング：タイトル＋問題提起
→ 読者が「自分のことだ」と感じる悩みを言語化
【STEP2：共感する】「この人はわかってくれる」
→ 対応ライティング：共感ストーリー
→ 「私も同じ状態から始めた」という体験談
【STEP3：期待する】「これなら解決できるかも」
→ 対応ライティング：解決策の提示＋得られる成果
→ 具体的な数字で「変化後の未来」を見せる
【STEP4：納得する】「この内容なら価値がある」
→ 対応ライティング：商品内容の詳細＋特典紹介
→ 目次を見せて「これだけ学べる」と感じさせる
【STEP5：決断する】「よし、買おう」
→ 対応ライティング：行動の後押し＋価格の納得感
→ 「この価格でこれだけ得られるなら安い」と感じさせる

💰 この章で得られる効果

4-1. 無料記事の3つの役割

📝 無料記事の役割
【役割1：信頼構築】
「この人は有益な情報を発信している」と思ってもらう
→ 無料で価値を提供することで、有料記事への信頼が生まれる
【役割2：集客】
「この人の他の記事も読みたい」と思ってもらう
→ 検索やSNSから流入した読者を、他の記事に誘導する
【役割3：有料記事への導線】
「もっと詳しく知りたい」と思ってもらう
→ 無料記事の最後に「さらに詳しくはこちら」と有料記事に誘導
4-2. 無料記事の設計パターン

💡 無料記事の「量」の目安
【有料記事公開前に】無料記事3～5本
→ 「この人は継続的に有益な発信をしている」という印象
【有料記事公開後】週に1本ペース
→ 常に新しい読者を集める"入口"を作り続ける
【ポイント】
無料記事は「消耗品」ではなく「集客装置」です。
一度書いた記事は、ずっと検索経由で読者を連れてきてくれます。
4-3. 導線設計の全体図
🗺️ 「集客→販売」の導線マップ
━━━━━━ 導線の全体像 ━━━━━━
【入口】
　SNS投稿 → note無料記事 → note有料記事
　Google検索 → note無料記事 → note有料記事
　note内検索 → note無料記事 → note有料記事
【各ポイントの役割】
　SNS投稿：興味を引く（「気になる」）
　note無料記事：信頼を築く（「この人良い」）

6-1. Brain（ブレイン）の特徴と活用法

🧠 Brainの最大の武器：アフィリエイト機能
【アフィリエイトとは】
購入者があなたの商品を紹介して、そこから売れたら購入者にも報酬が入る仕組み。
【何が起きるか】
① あなたが商品を販売
② 購入者Aが「これ良かったよ」とSNSで紹介
③ Aのリンク経由でBが購入
④ Aに紹介報酬が入る（価格の最大50%）
【メリット】
・購入者が「口コミするインセンティブ」がある
・自動的に口コミが広がる「バイラル型」の集客が可能
・自分で集客しなくても売れる可能性がある
6-2. Tips（ティップス）の特徴

📌 Tipsの特徴
  • 基本的なBrainと似たプラットフォーム
  • アフィリエイト機能あり
  • 手数料14%（Brainの12%よりやや高い）
  • 比較的新しいプラットフォームで競合が少ない
【使い分けの目安】
  • Brain：アフィリエイトで拡散を狙う商品
  • Tips：Brainと併用して露出を増やす
6-3. BOOTHの特徴と活用法

🏪 BOOTHの特徴
  • ダウンロード販売に特化（PDF、テンプレート等）
  • 手数料が安い（5.6%+22円/件）
  • ピクシブ（pixiv）ユーザーとの親和性が高い
  • クリエイター向けのイメージが強い
【BOOTHが向いている商品】
  • PDF教材（テンプレート集、チェックリスト）
  • Notionテンプレート
  • スプレッドシートテンプレート
  • プロンプト集（テキストファイル）

1-4. STP・4P・AIDMAを「ライティング視点」で理解する
はじめに：なぜライターがマーケティングを学ぶのか
「マーケティング？ライターには関係ないでしょ」
もしそう思っているなら、この章で考えが変わります。
結論から言います。マーケティングを知らないライターは、永遠に「作業者」のままです。
なぜなら、クライアントが本当に求めているのは「文章」ではなく「成果」だからです。商品が売れること、問い合わせが増えること、ブランドが認知されること。これらの成果を出すためには、マーケティングの知識が不可欠です。

マーケティングを知らないライターの限界
クライアント：「40代女性向けのダイエットサプリの記事をお願いします」
マーケティングを知らないライター
  • →「ダイエットサプリ」について調べて記事を書く
  • → 誰にでも当てはまる一般的な内容になる
  • → クライアントから「もっとターゲットに刺さる内容に」と修正依頼
  • → どう修正すればいいかわからない
マーケティングを知っているライター
  • →「40代女性」のペルソナを設定（STP）
  • → 競合との差別化ポイントを確認（4P）
  • → 購買心理に沿った構成を設計（AIDMA）
  • → 最初から「刺さる」記事を書ける
  • → クライアントから「さすがです！」と高評価
この差が、単価1文字0.5円のライターと、単価1文字5円のライターを分けます。
この章では、マーケティングの基本フレームワーク「STP」「4P」「AIDMA」を、ライティングに直結する形で解説します。
難しそうに見えますが、安心してください。一度理解すれば、すべてのライティングで使える「武器」になります。
第1章：STP分析 ― 「誰に書くか」を決める技術
STP分析とは、マーケティングの最も基本的なフレームワークの一つです。ライターにとってのSTPは、「誰に向けて書くかを明確にする技術」です。
1-1. STPとは何か
STPは、以下の3つの頭文字を取ったものです。

STPの3要素
S = Segmentation（セグメンテーション）: 市場を細分化する。「ダイエットに興味がある人」を、さらに細かく分ける。
T = Targeting（ターゲティング）: 狙う市場を決める。細分化した中から、最も狙うべきセグメントを選ぶ。
P = Positioning（ポジショニング）: 競合との違いを明確にする。選んだターゲットに対して、どんな立ち位置で訴求するか決める。
これをライティングに置き換えると、こうなります：
  • Segmentation → 読者を細かく分類する
  • Targeting → 誰に向けて書くか決める
  • Positioning → どんな切り口で書くか決める
1-2. Segmentation（セグメンテーション）― 読者を分類する
「ダイエットに興味がある人」と一口に言っても、実は様々な人がいます。

ダイエット市場のセグメンテーション例
年代で分ける
  • 20代：美容目的、見た目重視
  • 30代：産後ダイエット、体型維持
  • 40代：健康目的、代謝低下への対応
  • 50代以上：生活習慣病予防、医師の指導
悩みで分ける
  • 運動する時間がない人
  • 食事制限が続かない人
  • リバウンドを繰り返している人
  • 特定の部位（お腹、太もも等）が気になる人
ライフスタイルで分ける
  • 忙しい会社員
  • 子育て中の主婦
  • 在宅ワーカー
  • 健康意識の高い人
このように細分化することで、「誰に」書くべきかが明確になります。

セグメンテーションしないとどうなるか
「ダイエットに興味がある人」全員に向けて書こうとすると…
  • → 20代の悩みも、50代の悩みも、すべてカバーしようとする
  • → どの年代にも「ちょっと違う」と感じられる
  • → 結局、誰にも刺さらない文章になる
これが「読まれない文章」の根本原因です。
1-3. Targeting（ターゲティング）― 狙う読者を決める
セグメンテーションで分類したら、次は「誰を狙うか」を決めます。
ここで重要なのは、「絞る勇気」です。「みんなに読んでもらいたい」という欲を捨て、一つのセグメントに絞ります。

ターゲティングの判断基準
  • 市場規模：十分な数の読者がいるか
  • 到達可能性：そのターゲットに情報を届けられるか
  • 競合状況：競合が少ない、または差別化できるか
  • 自社（クライアント）の強み：その層に訴求できる強みがあるか

ターゲット設定の具体例
商品情報
  • クライアント：ダイエットサプリメーカー
  • 商品の特徴：代謝アップ成分配合、運動なしでOK
ターゲティングの思考プロセス
  • 20代 → 運動や食事制限を頑張れる層が多い。「運動なし」は刺さりにくい
  • 30代 → 産後ダイエット需要。時間がない層に「運動なし」は刺さる
  • 40代 → 代謝低下が深刻。「代謝アップ」が最も刺さる
  • 50代以上 → 医師の指導を重視。サプリだけでは不安を感じる層が多い
結論
ターゲットは「40代・働く女性・運動する時間がない層」に決定。
ターゲットが決まると、文章の方向性が一気に明確になります。
1-4. Positioning（ポジショニング）― 競合との違いを出す
同じターゲットを狙う競合は、必ず存在します。その中で選ばれるには、「この商品（記事）ならでは」の立ち位置を明確にする必要があります。

ポジショニングの考え方
競合A「1ヶ月で-10kg！劇的ダイエットサプリ」→ 短期間で大きな効果を訴求
競合B「医師監修の安心サプリ」→ 安全性・信頼性を訴求
競合C「芸能人愛用のプレミアムサプリ」→ ステータス・高級感を訴求
自社「40代の体に合わせた、無理しない代謝改善サプリ」→ 年代特化・継続しやすさを訴求
競合が「劇的な効果」「安心」「高級感」で争っている中、「40代特化」「無理しない」という独自のポジションを取る。
ポジショニングが決まると、文章で「何を強調すべきか」が明確になります。
同じターゲットを狙う競合は、必ず存在します。その中で選ばれるには、「この商品（記事）ならでは」の立ち位置を明確にする必要があります。

ポジショニングの考え方
【競合A】「1ヶ月で-10kg！劇的ダイエットサプリ」
→ 短期間で大きな効果を訴求
【競合B】「医師監修の安心サプリ」
→ 安全性・信頼性を訴求
【競合C】「芸能人愛用のプレミアムサプリ」
→ ステータス・高級感を訴求
【自社のポジショニング】
「40代の体に合わせた、無理しない代謝改善サプリ」
→ 年代特化・継続しやすさを訴求
競合が「劇的な効果」「安心」「高級感」で争っている中、「40代特化」「無理しない」という独自のポジションを取る。

ポジショニングマップの考え方
  • 縦軸：「効果の強さ（劇的 ↔ 穏やか）」
  • 横軸：「ターゲット（全年代 ↔ 年代特化）」
  • 自社の位置：「穏やか × 40代特化」＝ 競合が少ないポジション
ポジショニングが決まると、文章で「何を強調すべきか」が明確になります。
1-5. STPをライティングに活かす実践法
STPを理解したら、実際のライティングにどう活かすか見ていきましょう。

STP活用の3ステップ
1
クライアントにSTPを確認する: 「ターゲットは誰ですか？」「競合との違いは何ですか？」と質問する。明確な答えがなければ、一緒に考えることを提案する。
2
ペルソナを作成する: ターゲットを、一人の具体的な人物像に落とし込む。名前、年齢、職業、悩み、理想の状態などを設定する。
3
ペルソナに向けて書く: 「この人に向けて書いている」という意識を常に持つ。迷ったら「この人なら、どう感じるか？」と自問する。

ペルソナシートの例
基本情報
  • 名前：山田佳子（仮名）
  • 年齢：43歳
  • 職業：事務職（正社員）
  • 家族：夫、中学生の娘
  • 年収：400万円（世帯年収800万円）
悩み
  • 40歳を過ぎてから、体重が落ちにくくなった
  • 仕事と家事で忙しく、運動する時間がない
  • 食事制限を何度も試したが、続かない
  • 健康診断で「要注意」が増えてきた
  • お気に入りの服が入らなくなった
理想の状態
  • 無理なく、健康的に痩せたい
  • 家族に「きれいになったね」と言われたい
  • 自分に自信を持ちたい
情報収集の方法
  • Instagram、YouTube
  • ママ友からの口コミ
  • 健康系のWebメディア
購入の決め手
  • 口コミ評価が高い
  • 安全性が確認できる
  • 価格が続けられる範囲
このペルソナを設定すると、「山田佳子さんに向けて書く」という意識で文章が書けます。結果として、ターゲットに「刺さる」文章になります。
第2章：4P分析 ― 「何を伝えるか」を決める技術
4P分析は、マーケティングミックスとも呼ばれる、商品・サービスの訴求ポイントを整理するフレームワークです。ライターにとっての4Pは、「文章で何を伝えるべきかを整理する技術」です。
2-1. 4Pとは何か
4Pは、以下の4つの頭文字を取ったものです。

4Pの4要素
Product（製品）:
何を売るのか。商品・サービスの特徴、機能、品質、デザインなど。
Price（価格）:
いくらで売るのか。価格設定、割引、支払い方法など。
Place（流通）:
どこで売るのか。販売チャネル、店舗、オンラインなど。
Promotion（販促）:
どう知らせるのか。広告、PR、キャンペーンなど。
これをライティングに置き換えると、こうなります：
  • Product → 商品の「何を」伝えるか
  • Price → 価格の「見せ方」をどうするか
  • Place → 読者が「どこで」行動するか
  • Promotion → 「どんな訴求」で惹きつけるか
2-2. Product（製品）― 何を伝えるか
商品 Features... ベネフィット

特徴とベネフィットの違い
特徴（Feature）
商品が「持っている」もの。成分、機能、スペックなど。
ベネフィット（Benefit）
顧客が「得られる」もの。結果、変化、感情など。

特徴→ベネフィット変換の例
ビタミンC 1,000mg配合 → 肌のハリ・ツヤがアップ、若々しい印象に
24時間サポート対応 → 深夜でも安心、いつでも相談できる
重量わずか500g → 持ち運びラクラク、外出先でも使える
AI自動分析機能 → 専門知識なしでもプロ並みの分析ができる
ライティングでは、「特徴」だけでなく「ベネフィット」まで伝えることが重要です。

特徴だけを並べた文章の問題点
「このサプリには、ビタミンC、ビタミンE、コエンザイムQ10が配合されています。」
読者の反応：「で？それが私に何の関係があるの？」
→ 特徴だけでは、読者は「自分ごと」として捉えられない。
2-2. Product（製品）― 何を伝えるか
商品の特徴を整理し、「何を伝えるべきか」を決めます。重要なのは、「特徴」と「ベネフィット」を分けて考えることです。

特徴とベネフィットの違い
  • 特徴（Feature）：商品が「持っている」もの。成分、機能、スペックなど。
  • ベネフィット（Benefit）：顧客が「得られる」もの。結果、変化、感情など。

特徴→ベネフィット変換の例
特徴：ビタミンC 1,000mg配合
→ ベネフィット：肌のハリ・ツヤがアップ、若々しい印象に
特徴：24時間サポート対応
→ ベネフィット：深夜でも安心、いつでも相談できる
特徴：重量わずか500g
→ ベネフィット：持ち運びラクラク、外出先でも使える
特徴：AI自動分析機能
→ ベネフィット：専門知識なしでもプロ並みの分析ができる

特徴だけを並べた文章の問題点
「このサプリには、ビタミンC、ビタミンE、コエンザイムQ10が配合されています。」
読者の反応：「で？それが私に何の関係があるの？」
→ 特徴だけでは、読者は「自分ごと」として捉えられない。
ライティングでは、「特徴」だけでなく「ベネフィット」まで伝えることが重要です。
2-3. Price（価格）― 価格の見せ方
同じ価格でも、見せ方によって印象は大きく変わります。ライターは、価格を「高い」と感じさせない工夫を文章で表現する必要があります。

価格の見せ方テクニック
テクニック1：分割して見せる: 「月額3,000円」→「1日たった100円」
テクニック2：比較で見せる: 「ジム会費1万円/月の代わりに、このサプリ3,000円/月」
テクニック3：価値で見せる: 「3,000円で、10万円相当のノウハウを公開」
テクニック4：投資として見せる: 「この3,000円が、あなたの健康と自信を取り戻します」
テクニック5：リスクを下げる: 「30日間返金保証付き。効果を感じなければ全額返金」

価格訴求の文章例
悪い例
「価格は月額3,000円です。」
良い例
「1日あたり、たったの100円。
缶コーヒー1本分の投資で、20代の頃のような『燃える体』が手に入ります。
しかも、初回は50%OFFの1,500円。
30日間の返金保証付きなので、まずは1ヶ月、リスクなしでお試しいただけます。」
2-4. Place（流通）― どこで行動させるか
読者に「どこで」「どうやって」行動してもらうかを明確にします。ライティングでは、「次のアクション」を具体的に示すことが重要です。

行動導線の書き方
曖昧な導線
「興味のある方は、お問い合わせください。」
→ どこに？どうやって？わからない。
明確な導線
「今すぐ、下の青いボタンをクリックしてください。申込フォームが開きます。お名前とメールアドレスを入力するだけ。3分で完了します。」
→ 具体的で迷わない。
読者に「どこで」「どうやって」行動してもらうかを明確にします。ライティングでは、「次のアクション」を具体的に示すことが重要です。

行動導線の書き方
【曖昧な導線】
「興味のある方は、お問い合わせください。」
→ どこに？どうやって？わからない。
【明確な導線】
「今すぐ、下の青いボタンをクリックしてください。申込フォームが開きます。お名前とメールアドレスを入力するだけ。3分で完了します。」
→ 具体的で迷わない。

Place設計のポイント
  • 販売チャネルを明記する：「公式サイト限定」「Amazon限定」など
  • 行動のハードルを下げる：「3分で完了」「入力は3項目だけ」
  • 次のステップを一つに絞る：複数の行動を求めない
  • 導線を途切れさせない：LP→フォーム→決済→完了まで一気通貫で設計する
2-5. Promotion（販促）― どんな訴求で惹きつけるか
どんな切り口で商品を訴求するかを決めます。訴求軸を明確にすることで、文章全体の方向性が決まります。

主な訴求軸
機能訴求: 「業界最高レベルの○○機能を搭載」
価格訴求: 「今だけ50%OFF」「業界最安値」
品質訴求: 「職人が一つひとつ手作り」「厳選素材のみ使用」
実績訴求: 「累計100万個突破」「顧客満足度98%」
権威訴求: 「医師監修」「有名人愛用」
限定訴求: 「先着100名様限定」「今日だけの特別価格」
感情訴求: 「家族の笑顔を守る」「自分へのご褒美に」

訴求軸を決めるときの考え方
  • 競合が「機能訴求」で争っている場合 → 同じ土俵で戦っても埋もれる → 「感情訴求」や「実績訴求」など、別の軸を選ぶ
  • ターゲットが「価格」を重視している場合 → 「価格訴求」を軸にする → ただし、安さだけでなく「コスパ」として見せる
  • 商品に明確な強みがある場合 → その強みに合った訴求軸を選ぶ → 例：医師監修なら「権威訴求」
2-6. 4Pをライティングに活かす実践法
4Pを整理したら、それを文章に落とし込みます。

4P整理シートの例
Product（製品）
  • 特徴：代謝アップ成分配合、1日1粒でOK
  • ベネフィット：運動なしで脂肪燃焼、忙しくても続けられる
Price（価格）
  • 通常価格：3,000円/月
  • 初回価格：1,500円（50%OFF）
  • 見せ方：「1日50円で理想の体型へ」
Place（流通）
  • 販売チャネル：公式サイト限定
  • 行動導線：LP→申込フォーム→決済→商品発送
Promotion（販促）
  • 訴求軸：「40代特化」×「無理しない」
  • キャンペーン：初回50%OFF、30日間返金保証
このシートを元に文章を書くと、伝えるべきことが整理された状態で書き始められます。
4Pを整理したら、それを文章に落とし込みます。

4P整理シートの例
Product（製品）
  • 特徴：代謝アップ成分配合、1日1粒でOK
  • ベネフィット：運動なしで脂肪燃焼、忙しくても続けられる

4P活用のポイント
  • Productは「特徴」ではなく「ベネフィット」で伝える
  • Priceは「分割・比較・価値・投資・リスク軽減」で見せる
  • Placeは「次の一歩」を一つだけ明確に示す
  • Promotionは競合と差別化できる訴求軸を選ぶ
第3章：AIDMA ― 「どう伝えるか」を決める技術
AIDMAは、消費者の購買心理プロセスを表すモデルです。1-2でも触れましたが、ここではより深く、ライティングへの活用法を解説します。ライターにとってのAIDMAは、「読者の心を動かす順番を設計する技術」です。
3-1. AIDMAとは何か（復習）
AIDMAは、以下の5つの段階で構成されています。

AIDMAの5段階
01
A = Attention（注意）: まず、注意を引く。存在を認知してもらう。
02
I = Interest（関心）: 興味を持たせる。「もっと知りたい」と思わせる。
03
D = Desire（欲求）: 欲しいと思わせる。「これが必要だ」と感じさせる。
04
M = Memory（記憶）: 記憶に残す。後で思い出してもらう。
05
A = Action（行動）: 行動させる。購入、申込、問い合わせなどを促す。

3-2. 各段階の詳細とライティング手法
【Attention】注意を引く
読者の注意を引かなければ、どんなに良い内容も読まれません。最初の3秒で「続きを読みたい」と思わせることが勝負です。

Attentionを獲得するテクニック
  • 数字を使う：「97%の人が知らない」「3分でわかる」
  • 疑問を投げかける：「なぜ、あなたは痩せないのか？」
  • 意外性を出す：「ダイエットの常識は、間違いだらけ」
  • ターゲットを名指しする：「40代女性のあなたへ」
  • ベネフィットを予告する：「この記事を読めば、○○がわかります」

Attention獲得の文章例
タイトル・見出し
「40歳を過ぎたら、ダイエットの常識は捨ててください」
リード文
あなたは、まだ「食べる量を減らせば痩せる」と思っていませんか？実は、40代以降の体には、それが逆効果になることがあります。この記事では、40代の体に合った「新しいダイエット法」をお伝えします。

【Interest】関心を持たせる
注意を引いたら、次は「興味を持続」させます。「もっと知りたい」「続きが気になる」と思わせることが目標です。

Interestを維持するテクニック
  • 好奇心を刺激する：「その秘密は、意外なところにありました」
  • 共感を示す：「私も同じ悩みを抱えていました」
  • 権威を借りる：「ハーバード大学の研究によると…」
  • 問題を深掘りする：「なぜ、これまでの方法では失敗したのか」
  • ストーリーを語る：「3ヶ月前の私は…」

Interest維持の文章例
「なぜ、40代になると痩せにくくなるのか？」
その答えは、「基礎代謝の低下」にあります。20代の頃、何もしなくても消費していたカロリー。
それが、40代では約200kcal/日も減少しています。
つまり、同じ食事をしていても、毎日おにぎり1個分、余分に太るのです。
「でも、代謝を上げるなんて、難しいんじゃ…」そう思いますよね。
私もそう思っていました。でも、ある方法を知ってから、その考えは変わりました。
【Desire】欲しいと思わせる
関心を持った読者に、「これが欲しい」「これが必要だ」と感じさせます。ここでは「ベネフィット」を徹底的に伝えることが重要です。

Desireを喚起するテクニック
  • Before/Afterを見せる：変化を具体的にイメージさせる
  • 感情に訴える：「自信を取り戻す」「家族に褒められる」
  • 社会的証明を示す：「10万人が選んだ」「口コミ評価4.8」
  • 限定性を出す：「今だけ」「先着100名」
  • リスクを下げる：「返金保証付き」「初回無料」

【Memory】記憶に残す
読者がすぐに行動しない場合でも、後で思い出してもらう必要があります。「覚えやすいキーワード」や「印象的なフレーズ」を残すことが重要です。

Memoryを強化するテクニック
  • キャッチフレーズを作る：「40代からの、燃える体づくり」
  • 繰り返す：重要なメッセージを文章内で繰り返す
  • 比喩を使う：「体の中の『エンジン』を入れ替える」
  • 数字で印象づける：「3ヶ月で-5kg」「1日100円」
  • 感情に紐づける：「家族の笑顔のために」

Memory強化の文章例
覚えておいてください。
40代のダイエットは、「減らす」ではなく「燃やす」。
食べる量を減らすのではなく、燃やす力を取り戻す。これが、40代からの新常識です。
「燃やすダイエット」、今日から始めてみませんか？
【Action】行動させる
最後に、具体的な行動を促します。「何を」「どこで」「どうやって」行動すればいいか、迷わせないことが重要です。

Actionを促すテクニック
  • 具体的な行動を指示：「下のボタンをクリック」
  • 簡単さをアピール：「3分で完了」「入力はたったの3項目」
  • 緊急性を出す：「本日23:59まで」「残り12個」
  • 不安を取り除く：「いつでも解約OK」「返金保証付き」
  • 背中を押す：「今日が、新しい自分への第一歩です」

Action促進の文章例

下の緑色のボタンをクリックしてください。

3-3. AIDMAの発展形：AISAS
インターネット時代に合わせて、AIDMAを発展させた「AISAS」というモデルもあります。

AISASの5段階
1
A = Attention（注意）: 注意を引く（AIDMAと同じ）
2
I = Interest（関心）: 興味を持たせる（AIDMAと同じ）
3
S = Search（検索）: 検索して調べる（ネット時代特有）
4
A = Action（行動）: 購入する
5
S = Share（共有）: SNSで共有する、口コミを書く
Web記事やSNS向けのライティングでは、「Search」と「Share」を意識した設計が重要です。

AISASを意識したライティング
Search対策
  • 読者が検索しそうなキーワードを文章に含める
  • 「よくある質問」や「比較」など、検索ニーズに応える内容を入れる
Share促進
  • 「シェアしたくなる」要素を入れる（驚きの数字、共感できる内容）
  • 「この記事が役に立ったら、シェアしてください」と促す
  • SNSシェアボタンの位置を意識した文章構成にする
第4章：3つのフレームワークを統合する
STP、4P、AIDMAは、それぞれ独立したものではありません。3つを統合して使うことで、「誰に」「何を」「どう伝えるか」が一貫した文章が書けます。
4-1. 統合の考え方

3フレームワークの役割
STP → 「誰に」を決める:
ターゲットを明確にし、ペルソナを設定する
4P → 「何を」を決める:
伝えるべき内容（製品、価格、場所、訴求）を整理する
AIDMA → 「どう」を決める:
伝える順番・流れを設計する

統合の流れ
Step 1【STP】ターゲットを決める:
「40代・働く女性・運動する時間がない層」に決定
Step 2【4P】伝える内容を整理する:
Product：代謝アップサプリ、1日1粒 / Price：初回1,500円（1日50円）/ Place：公式サイト限定 / Promotion：「40代特化」×「無理しない」
Step 3【AIDMA】伝える順番を設計する:
A（注意）：「40歳を過ぎたら、ダイエットの常識は捨ててください」
I （関心）：基礎代謝低下のメカニズムを説明
D（欲求）：口コミ、Before/After、ベネフィット
M（記憶）：「燃やすダイエット」というキーワード
A（行動）：初回50%OFF、今すぐ申込

まとめ：フレームワークは「武器」になる
この章では、STP、4P、AIDMAの3つのマーケティングフレームワークを、ライティング視点で学びました。重要なポイントを振り返りましょう。

STP ― 「誰に書くか」を決める
  • Segmentation：市場を細分化する
  • Targeting：狙うセグメントを選ぶ
  • Positioning：競合との違いを明確にする
→ ペルソナを設定し、「その人に向けて」書く

4P ― 「何を伝えるか」を決める
  • Product：特徴とベネフィットを整理する
  • Price：価格の見せ方を工夫する
  • Place：行動導線を明確にする
  • Promotion：最も効果的な訴求軸を選ぶ

AIDMA ― 「どう伝えるか」を決める
  • Attention：注意を引く
  • Interest：関心を持たせる
  • Desire：欲しいと思わせる
  • Memory：記憶に残す
  • Action：行動させる
これらのフレームワークは、最初は「面倒」に感じるかもしれません。しかし、使い続けるうちに「思考の型」として身についていきます。
フレームワークを使いこなせるようになると、クライアントから「この人は違う」と思われるライターになれます。
次の章では、「ChatGPTに狙った文章を書かせる指示の出し方」を学びます。ここまでの知識を活かして、さらに実践的なスキルを身につけましょう。

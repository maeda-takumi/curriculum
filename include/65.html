2-3. Amazonアソシエイト登録手順

📝 登録手順【完全版】
【Step1】Amazonアソシエイトにアクセス
　→ https://affiliate.amazon.co.jp
　→ 「登録」をクリック
【Step2】Amazonアカウントでログイン
　→ 既存のAmazonアカウントを使用（なければ新規作成）
【Step3】アカウント情報を入力
→ 氏名、住所、電話番号
→ 報酬の受取人情報
【Step4】Webサイト・アプリ情報を入力
→ 自分のブログ、SNSのURLを登録
→ X、Instagram、YouTubeなども登録可能
→ 最低1つは必要
【Step5】プロフィールを入力
→ サイトのジャンル、集客方法などを選択
【Step6】本人確認（電話番号認証）
→ 電話番号を入力→SMSまたは電話で認証
【Step7】登録完了
→ ただし「仮承認」状態。180日以内に3件の売上が必要。
2-4. 審査を通過するコツ

✅ 審査通過のポイント
【ポイント1】登録するサイト・SNSにコンテンツがあること
→ ブログなら最低10記事以上
→ SNSなら投稿がある程度あること
【ポイント2】180日以内に3件の売上を作る
→ 自分で購入してもOK（自己アフィリは禁止だが、純粋な購入は可）
→ 知人に頼んで購入してもらうのも有効
【ポイント3】違反コンテンツがないこと
→ アダルト、暴力、著作権侵害がないこと
【ポイント4】サイトに運営者情報があること
→ プロフィール、プライバシーポリシー、お問い合わせ先
【もし審査に落ちたら】
→ もしもアフィリエイト経由でAmazonを扱う（後述）
2-5. Amazonリンクの作成方法

🔗 アフィリエイトリンクの作り方
【方法1】アソシエイト・ツールバーを使う
1. Amazonにログインした状態で商品ページを開く
2. ページ上部の「アソシエイトツールバー」が表示される
3. 「テキスト」「画像」「テキストと画像」からリンクタイプを選択
4. 生成されたリンクをコピーして使用
【方法2】アソシエイト・セントラルで作成
1. アソシエイト・セントラルにログイン
2. 「商品リンク」→商品を検索
3. リンクを作成→コピー
【方法3】短縮URL（amzn.to）を使う
→ ツールバーで「短縮URL」を選択
→ SNS投稿に便利

3-2. もしもアフィリエイトのメリット

✨ もしもアフィリエイトを使う理由
  • 【メリット1】Amazon審査に落ちても使える
  • Amazonアソシエイト本家より審査が緩い
  • 本家で落ちた人も、もしも経由なら通る場合が多い
  • 【メリット2】W報酬制度で10%ボーナス
  • 月の報酬に対して10%のボーナスが上乗せ
  • 1万円の報酬なら、1,000円プラスで11,000円
  • 【メリット3】かんたんリンクが便利
  • Amazon、楽天、Yahoo!の3つのボタンを一度に表示
  • ユーザーが好きなショップを選べる
  • 【メリット4】振込手数料が無料
  • どの銀行でも振込手数料0円
3-3. もしもアフィリエイト登録手順

📝 登録手順【完全版】
  • 【Step1】もしもアフィリエイトにアクセス
  • https://af.moshimo.com
  • 「無料でメディア会員登録」をクリック
  • 【Step2】メールアドレスを登録
  • 仮登録メールが届く→URLをクリック
  • 【Step3】会員情報を入力
  • 氏名、住所、生年月日など
  • 【Step4】サイト情報を登録
  • 自分のブログ、SNSのURL
  • サイトのカテゴリー、月間PV数（大まかでOK）
  • 【Step5】登録完了
  • これで「もしも」への登録は完了
  • 【Step6】Amazon、楽天、Yahoo!と提携申請
  • 管理画面から「プロモーション検索」
  • 「Amazon」「楽天」「Yahoo!」を検索して提携申請
  • 審査結果を待つ（即時〜数日）
3-4. かんたんリンクの作り方

🔗 かんたんリンクの作成手順
  • 【Step1】管理画面から「かんたんリンク」を選択
  • 上部メニュー「メディア運営ツール」→「かんたんリンク」
  • 【Step2】商品を検索
  • 商品名やキーワードで検索
  • 紹介したい商品を選択
  • 【Step3】リンクをカスタマイズ
  • ボタンの表示（Amazon、楽天、Yahoo!）を選択
  • 画像サイズを選択
  • 【Step4】コードをコピー
  • 「HTMLソースを1行にする」にチェック
  • コードをコピーしてブログに貼り付け
  • 【SNSでの使い方】
  • SNSでは「かんたんリンク」は使えない（HTMLが表示できないため）
  • 代わりに「商品リンク」で個別URLを取得して使用
3-5. 楽天・Amazon両方を紹介する方法

💡 両方紹介のテンプレート
  • 【ブログ記事の場合】
  • 記事内に「かんたんリンク」を設置
  • ユーザーが好きな方を選べる
  • 【SNS投稿の場合】
  • 投稿文に両方のリンクを記載
  • 例：
  • 「〇〇がめっちゃ良かった！」
    → 楽天：（楽天ROOMのURL）
    → Amazon：（Amazonリンク）
  • 【プロフィールリンクの場合】
  • lit.linkやlinktreeで複数リンクをまとめる
  • 「楽天ROOM」「Amazonおすすめ」など分けて設置

📝 登録手順【完全版】
【Step1】A8.netにアクセス → https://www.a8.net → 「アフィリエイトを始めたい方」をクリック
【Step2】メールアドレスを登録 → 仮登録メールが届く→URLをクリック
【Step3】会員情報を入力 → 氏名、住所、生年月日、電話番号
【Step4】メディア（サイト）情報を入力 → サイト名、URL、カテゴリー、月間PV数 → ※サイトがなくても「ファンブログ」で登録可能
【Step5】口座情報を入力 → 報酬振込先の銀行口座
【Step6】登録完了 → 審査なしですぐに利用開始 → 案件への提携申請は個別に行う
4-4. 案件の探し方・提携申請

🔍 案件の探し方
【方法1】カテゴリーから探す → 「プログラム検索」→カテゴリーを選択 → 金融、美容、健康、転職など
【方法2】キーワードで検索 → 「プログラム検索」→キーワード入力 → 例：「クレジットカード」「転職」「VOD」
【方法3】ランキングから探す → 「ランキング」で人気案件をチェック → 売れている案件は成約しやすい
【提携申請の流れ】1. 紹介したい案件を見つける
2. 「提携申請する」ボタンをクリック
3. 審査結果を待つ（即時承認〜数日）
4. 承認されたらリンクを作成して紹介開始
4-5. 高単価案件で稼ぐコツ

💰 高単価案件で成約を取るコツ
【コツ1】「体験談」を書く → 実際に使った感想は最強のコンテンツ → 「セルフバック」で自分で申し込んでから紹介
【コツ2】「比較記事」を書く → 「〇〇 おすすめ5選」「〇〇 vs △△ 比較」 → 複数の案件を比較して、読者に選んでもらう
【コツ3】「悩み解決」の切り口 → 「転職したいけど不安…」→転職サービスの紹介 → 「副業で稼ぎたい」→スキル講座の紹介
【コツ4】SNSからブログ記事へ誘導 → SNSで興味を引く→詳細はブログで→アフィリエイトリンク
4-6. セルフバックを活用する

🎁 セルフバックとは
【セルフバックとは】 → 自分で申し込んで報酬を得られる仕組み → A8.netの「セルフバック」コーナーで案件を探す
【おすすめのセルフバック案件】
  • クレジットカード発行（年会費無料のもの）→ 数千〜1万円
  • FX口座開設 → 5,000〜2万円
  • VOD無料トライアル → 数百〜2千円
  • 保険の無料相談 → 数千〜1万円
【メリット】
  • 報酬を得ながら、サービスを体験できる
  • 体験したサービスをレビュー記事にできる
  • 初月から数万円の収益を作れる可能性

5-4. 複数ASPに登録すべき理由

💡 複数ASP登録のメリット
【理由1】同じ案件でも報酬が違う場合がある
→ A8.netでは3,000円、afbでは3,500円、など
→ 比較して高い方を選べる
【理由2】ASPごとに独占案件がある
→ 「このASPでしか扱えない」案件がある
【理由3】担当者がつくと特別単価がもらえることも
→ 実績を積むと、通常より高い報酬がもらえる場合あり
【おすすめの組み合わせ】
  • 必須：A8.net、もしもアフィリエイト
  • 推奨：バリューコマース、afb
  • ジャンルに応じて：アクセストレード、その他

6-4. 「導線設計」の考え方

🔄 効果的な導線設計
【パターン1】SNS → 楽天ROOM / Amazon直接リンク
→ 即購入してもらいたい商品向け
→ 衝動買いしやすい低〜中価格帯の商品
【パターン2】SNS → ブログ記事 → アフィリエイトリンク
→ 比較検討が必要な商品・サービス向け
→ 高単価案件（金融、スクールなど）
→ ブログで詳しく説明→納得して申込
【パターン3】SNS → プロフィールリンク → 複数リンク
→ Instagramなど、投稿にリンクが貼れない場合
→ lit.link、linktreeで複数のリンクをまとめる
6-5. リンク管理の効率化

📋 リンク管理のコツ
【コツ1】スプレッドシートで一元管理
→ 商品名、楽天URL、AmazonURL、報酬率を記録
→ 使いたいときにコピペで取り出せる
【コツ2】よく使うリンクはテンプレ化
→ SNS投稿用のテンプレートを作っておく
→ 「楽天→〇〇、Amazon→〇〇」の形式
【コツ3】短縮URLを活用
→ Bitlyなどで短縮URLを作成
→ クリック数のトラッキングも可能
【コツ4】定期的にリンク切れをチェック
→ 販売終了した商品のリンクは修正

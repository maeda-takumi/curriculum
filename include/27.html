2-7. 単価が上がる人・上がらない人の分岐点
はじめに：いつまで「文字単価0.5円」を続けますか？
クラウドソーシングで案件を取れるようになった。記事も書けるようになった。 でも、いつまで経っても文字単価が上がらない…。そんな悩みを抱えていませんか？
同じタイミングで始めたのに、1年後に文字単価3円になる人と、いつまでも0.5円のままの人がいます。
その差は、才能ではありません。「やり方」の差です。
【単価が上がらない人】
  • 「実績を積めば自然と上がる」と思っている
  • 言われた仕事をこなすだけ
  • 低単価の案件を「数」でカバーしようとする
  • 単価交渉をしたことがない
  • 「自分にはまだ早い」と思っている
→ 1年経っても、2年経っても文字単価0.5円のまま
【単価が上がる人】
  • 「単価を上げる」ことを意識して動いている
  • クライアントに「価値」を提供している
  • 適切なタイミングで単価交渉をしている
  • より高単価の案件に挑戦している
  • 自分の「市場価値」を高める努力をしている
→ 半年〜1年で文字単価1円→2円→3円とステップアップ
この章では、単価が上がる人と上がらない人の「分岐点」を解説し、あなたが単価アップを実現するための具体的な方法をお伝えします。

フェーズ2の最後に、この先の「稼ぎ方」の方向性を決める重要な章です。
第1章：なぜ単価が上がらないのか
まず、単価が上がらない「本当の理由」を理解しましょう。
1-1. 「実績を積めば上がる」は幻想
多くの人が「実績を積めば自然と単価が上がる」と思っています。 これは半分正解、半分不正解です。
【現実】
  • 同じクライアントから同じ単価の仕事を100件こなしても、単価は「自動的には」上がらない
  • 「経験3年」「実績500件」でも、文字単価0.5円のまま消耗している人は多い
  • 実績は「あって当たり前」であり、それだけでは差別化にならない
【真実】
単価が上がるのは「実績を積んだから」ではなく、「価値を高める行動をしたから」
  • より高単価の案件に応募した
  • 単価交渉をした
  • 新しいスキルを身につけた
  • クライアントに不可欠な存在になった
→ 「待っていても」単価は上がらない
1-2. 低単価ループから抜け出せない理由
低単価の仕事から抜け出せない人には、共通のパターンがあります。
【パターン①：数で稼ごうとする】
「単価が低いなら、数をこなせばいい」
  • 時間がなくなり、スキルアップの余裕がない
  • いつまでも低単価から抜け出せない
  • 疲弊して続かなくなる
【パターン②：「まだ自分には早い」と思う】
「文字単価1円の案件？まだ自分には早い」「もっと実績を積んでから挑戦しよう」
  • いつまでも「そのとき」が来ない
  • 実際には、挑戦すれば通る案件も多い
  • チャンスを逃し続ける
【パターン③：居心地の良いところにとどまる】
「今のクライアントは楽だから、このままでいいか」「新しい案件を探すのは面倒」
  • 成長の機会を逃す
  • クライアント都合で切られたら終わり
  • スキルも収入も停滞
【パターン④：単価交渉を避ける】
「交渉したら嫌われるかも」「断られたら気まずい」
  • クライアントは「この単価でOKなんだ」と思う
  • 交渉しない限り、上がることはない
  • 何年経っても同じ単価

💡 4つのパターン まとめ
  • ① 数でカバー → 疲弊して終わる
  • ② 「まだ早い」思考 → チャンスを永遠に逃す
  • ③ 現状維持 → スキルも収入も止まる
  • ④ 交渉回避 → 単価は一生上がらない
→ どれか1つでも当てはまるなら、今すぐ意識を変えましょう
1-3. 単価は「言い値」で決まる
ここで重要な事実をお伝えします。
単価は「あなたがいくらで受けるか」で決まります。
クライアントが「この人は0.5円で受けてくれる」と思えば、ずっと0.5円のまま。 「この人は2円じゃないと受けてくれない」と思えば、2円を提示してきます。
【単価を決める要素】
  • ① あなたが「いくらで受けるか」を決める
  • ② クライアントが「この人にはいくら払う価値があるか」と判断する
  • ③ 市場の相場
→ ①と②は、あなたの行動で変えられる
第2章：単価が上がる人の「5つの共通点」
単価を上げていく人には、共通する行動パターンがあります。
2-1. 共通点①：高単価案件に「挑戦」している
単価が上がる人は、今の自分より少し上の案件に常に挑戦しています。
【挑戦の仕方】
現在：文字単価0.5円の案件を受注中
  • 挑戦①：文字単価0.8〜1円の案件に応募してみる
  • 挑戦②：「未経験可」「初心者OK」以外の案件にも応募
  • 挑戦③：条件の良い案件を見つけたら、ダメ元で応募
→ 「受かったらラッキー」くらいの気持ちで挑戦 → 意外と通ることも多い
【挑戦しない人の言い訳】
  • 「まだ実績が足りない」 → 実績は、挑戦しながら作るもの
  • 「落ちたら恥ずかしい」 → 誰も見ていない。気にするのは自分だけ
  • 「今の仕事で手一杯」 → 低単価の仕事を減らして、挑戦の時間を作る
2-2. 共通点②：「価値」を提供している
単価が上がる人は、「言われた仕事をこなす」だけでなく、「価値」を提供しています。
【「価値」の具体例】
① 期待以上のクオリティ
  • 指定より少し多い文字数
  • 読みやすい構成
  • 追加のリサーチ
② 手間の軽減
  • 修正が少ない
  • 確認事項を先回りで聞く
  • レギュレーションを完璧に守る
③ 信頼の提供
  • 納期より早く納品
  • 返信が早い
  • 安心して任せられる
④ プラスαの提案
  • 「こうした方がいいかも」という提案
  • SEO的な改善点の指摘
  • 読者目線でのアドバイス
2-3. 共通点③：単価交渉をしている
単価が上がる人は、適切なタイミングで単価交渉をしています。
【単価交渉の考え方】
  • 交渉しなければ、上がることはない
  • 適切な交渉は、むしろ好印象
  • 「この人は自分の価値を理解している」と思われる
  • 断られても、関係が壊れることは少ない
  • 交渉を重ねることで、交渉力も上がる
具体的な交渉方法は、第5章で詳しく解説します。
2-4. 共通点④：「専門性」を磨いている
単価が上がる人は、何かしらの「専門性」を持っています。
【専門性の例】
【ジャンルの専門性】
金融・投資 / 美容・コスメ / IT・テクノロジー / 医療・健康
【スキルの専門性】
SEOに強い / セールスライティング / 取材・インタビュー / 構成作成
【経験の専門性】
実際に転職を経験した / 育児中で子育て情報に詳しい / 元営業で顧客心理がわかる
「何でもできます」より「〇〇が得意です」の方が、高単価につながります。
2-5. 共通点⑤：「断る」ことができる
単価が上がる人は、低単価の案件を「断る」ことができます。
【断ることの重要性】
  • 低単価の案件を受け続けると、時間がなくなる
  • 時間がないと、スキルアップできない
  • スキルアップできないと、単価が上がらない
→ 低単価の案件を「断る」ことで、より良い案件に時間を使える
【断るタイミング】
  • 高単価の案件が増えてきたとき
  • 時間が足りなくなってきたとき
  • 明らかに割に合わない案件を提示されたとき
  • クライアントの対応が悪いとき
第3章：単価別「稼ぎ方」のステージ
単価によって、求められることや稼ぎ方が変わります。自分が今どのステージにいるかを理解しましょう。
3-1. ステージ1：文字単価0.3〜0.5円
【このステージの特徴】
  • 初心者が最初に通るステージ
  • 「誰でもできる」仕事が中心
  • タスク案件、マニュアル完備の案件
  • 実績作りの段階
【このステージですべきこと】
  • ① とにかく実績を作る：評価★5を積み上げる / プロフィールに書ける実績を増やす
  • ② 基本を身につける：納期厳守 / 丁寧なコミュニケーション / 指示通りに書く力
  • ③ 次のステージを見据える：1〜2ヶ月で卒業することを目指す / いつまでもここにとどまらない

✅ 卒業の目安
  • クラウドワークスでの実績 10〜20件
  • 評価 ★4.5以上
  • 基本的な記事が問題なく書ける
3-2. ステージ2：文字単価0.8〜1.5円
【このステージの特徴】
  • 「ある程度書ける人」が求められる
  • SEO記事、商品紹介記事など
  • 継続案件が増えてくる
  • 月3〜5万円が見えてくる
【このステージですべきこと】
  • ① SEOの基礎を学ぶ：キーワード選定の考え方 / 構成の作り方 / 読者の検索意図
  • ② 継続案件を獲得する：信頼を積み上げる / 「また頼みたい」と思われる対応
  • ③ 得意ジャンルを作る：1〜2個の得意分野を持つ / 専門性をアピールできるようになる

✅ 卒業の目安
  • 継続クライアントが 2〜3社
  • SEO記事が書ける
  • 構成から任されることがある
3-3. ステージ3：文字単価2〜3円
【このステージの特徴】
  • 「信頼できるライター」として認識される
  • 構成作成から任される
  • 専門性が評価される
  • 月10〜20万円が見えてくる
【このステージですべきこと】
  • ① 専門性を深める：特定ジャンルで「この人」と言われる存在に / 資格取得、学習を続ける
  • ② 構成力を磨く：構成から任される案件を増やす / SEOだけでなく、読者体験を考えた構成
  • ③ クライアントとの関係を深める：「外注」から「パートナー」へ / 提案、相談ができる関係に

⚠ このステージの注意点
  • このあたりで「頭打ち」になる人が多い
  • さらに上を目指すには、次のステージへの意識が必要
  • 「今のまま継続すれば安定」という油断が成長を止める
3-4. ステージ4：文字単価3円以上
【このステージの特徴】
  • 専門家、プロとして認識される
  • 指名で依頼が来る
  • 単価交渉が通りやすい
  • 月20万円以上が安定する
【このステージの人がやっていること】
  • ① 専門分野での実績・知名度：「この分野なら〇〇さん」と言われる / SNS、ブログでの発信
  • ② 付加価値の提供：取材、インタビューができる / ディレクションができる / コンサルティング的な提案ができる
  • ③ クライアントの事業に貢献：記事を通じて売上に貢献 / SEOで成果を出す / 数字で語れる実績
第4章：単価を上げる「7つの具体策」
単価を上げるための具体的なアクションを紹介します。
4-1. 策①：より高単価の案件に応募する
最もシンプルで効果的な方法です。
【実践方法】
  • 今の単価より「0.5円〜1円」高い案件を探す
  • 「未経験可」「初心者OK」以外にも応募する
  • 応募文で「なぜ自分が適任か」をアピールする
  • 落ちても気にしない。10件応募して1件通ればOK
【ポイント】
  • 「受かるかわからない」案件に挑戦する
  • 今の案件と並行して、チャレンジ枠を作る
  • 高単価案件に通ったら、低単価案件を減らす
4-2. 策②：既存クライアントに単価交渉する
すでに取引があるクライアントへの単価交渉は、成功率が高いです。
【交渉のタイミング】
  • 5〜10件以上の取引実績ができた
  • 継続的に高評価をもらっている
  • 担当範囲が広がった（構成作成も任されるなど）
  • 明らかに相場より低い単価で受けている
交渉メッセージ例
〇〇様
いつもお世話になっております。△△です。
継続的にお仕事をいただき、ありがとうございます。
1点、ご相談がございます。
現在の単価は〇円/文字でお受けしておりますが、今後〇円/文字にご調整いただくことは可能でしょうか。
これまで〇件の記事を納品し、レギュレーションも把握できております。引き続き、品質・納期を守って対応いたします。
ご予算もあるかと存じますので、難しければ現状のままでも問題ございません。ご検討いただけますと幸いです。よろしくお願いいたします。
4-3. 策③：得意ジャンルを作る
専門性があると、高単価の案件に通りやすくなります。
【得意ジャンルの作り方】
  • 本業・経験から選ぶ：仕事で詳しくなった分野 / 過去の経験が活かせる分野
  • 興味・関心から選ぶ：好きで調べている分野 / 時間を忘れて学べる分野
  • 需要から選ぶ：高単価案件が多い分野 / 専門性が求められる分野（金融、医療、IT、法律など）
【得意ジャンルを深める方法】
  • 関連する本を10冊読む
  • 関連する資格を取る
  • そのジャンルの案件を集中的に受ける
  • ブログやSNSで発信する
4-4. 策④：構成作成から受注する
「本文だけ」より「構成＋本文」の方が単価は高くなります。
【構成作成のメリット】
  • 単価が上がる（構成費が加算される）
  • 任される範囲が広がる
  • 「できるライター」として認識される
  • 自分の書きやすい構成で書ける
【構成を任されるためにすべきこと】
  • 構成案を「提案」してみる
  • 「構成から対応可能です」とアピール
  • SEO構成の作り方を学ぶ
4-5. 策⑤：直接契約を目指す
クラウドソーシング経由ではなく、直接契約することで単価アップが見込めます。
【直接契約のメリット】
  • 手数料がかからない（クラウドワークスは報酬の20%）
  • 単価交渉がしやすい
  • より深い関係が築ける

⚠ このステージの注意点
  • クラウドワークス上では直接契約の勧誘は規約違反
  • クライアントから「直接やり取りしませんか」と言われた場合のみ検討
  • 信頼関係ができてから。焦って動くとトラブルの原因になります
4-6. 策⑥：スキルを増やす
できることが増えると、単価アップの理由になります。
【単価アップにつながるスキル】
【ライティング関連】SEOライティング / セールスライティング / 取材・インタビュー / 構成作成
【周辺スキル】WordPress入稿 / 画像選定・作成 / SNS運用 / ディレクション
【スキルの身につけ方】
  • 実案件で経験を積む（少し安くても経験のために受ける）
  • 本、教材で学ぶ
  • できる人のやり方を観察する
4-7. 策⑦：実績を「見える化」する
実績があっても、伝わらなければ意味がありません。
【実績の見える化】
【プロフィール】「SEO記事100件以上執筆」「金融・投資分野が得意」「担当記事が検索1位を獲得」
【ポートフォリオ】公開可能な記事をジャンル別にまとめる / クライアントの許可を取って掲載
【提案文】「実績〇件」「承認率〇%」など具体的な数字を入れる / 関連する実績をアピール
第5章：単価交渉の「正しいやり方」
単価交渉は、多くの人が苦手とするポイントです。正しいやり方を身につけましょう。
5-1. 単価交渉への「心理的ハードル」を下げる
まず、単価交渉に対する考え方を変えましょう。
間違った考え方
  • 交渉したら嫌われる
  • 図々しいと思われる
  • 断られたら気まずい
  • 今の関係が壊れる
正しい考え方
  • ビジネスの場で交渉は当たり前
  • 適切な交渉はむしろプロ意識の表れ
  • 自分の価値を理解している人と思われる
  • 断られても関係が壊れることは稀
  • 交渉しなければずっと同じ単価のまま
単価交渉は「お願い」ではなく「ビジネス上の提案」です。自分の価値を正しく伝えることは、プロとして当然の行動です。
5-2. 交渉のタイミング
交渉に適したタイミング○
  • ①一定の実績ができたとき：同じクライアントと5〜10件以上の取引 / 継続的に高評価をもらっている
  • ②担当範囲が広がったとき：構成作成も任されるようになった / 新しいメディアの立ち上げを任された
  • ③クライアントから感謝されたとき：いつも助かっていますと言われた / 〇〇さんのおかげでと言われた
  • ④相場と明らかに乖離しているとき：同じレベルの仕事が他では倍の単価
  • ⑤継続依頼を打診されたとき
    最適なタイミング：今後も継続でお願いしたいと言われた / このタイミングで条件を相談しやすい
交渉を避けるべきタイミング✕
  • まだ2〜3件しか取引していない
  • 修正が多かった案件の直後
  • クライアントが明らかに忙しそうなとき
  • トラブルがあった後
5-3. 交渉の金額の決め方
金額の決め方
  • ①現状の1.2〜1.5倍を目安に：いきなり2倍は難しい / 段階的に上げていくイメージ
  • ②市場相場を調べる：同じようなレベルの案件の相場を確認 / 相場から大きく外れない金額
  • ③具体的な金額を提示する：少し上げてほしいは曖昧 / 〇円→△円にと明確に
金額の例
現状：文字単価1円
  • 提示①1.2円
    20%アップ→通りやすい
  • 提示②1.5円
    50%アップ→理由次第で可能
  • 提示③2円
    100%アップ→難しいが試す価値はある
5-4. 交渉メッセージのテンプレート

5-5. 交渉のコツ
コツ①理由を明確にする
  • なぜ単価アップをお願いするのか
  • どんな価値を提供しているか
  • 具体的な数字や実績を示す
コツ②退路を用意する
「難しければ現状のままで問題ございません」
  • →クライアントも断りやすい
  • →関係が壊れにくい
  • →断られた後も普通に仕事を続けられる
コツ③交渉決裂のリスクを受け入れる
  • 最悪この仕事を失っても良い覚悟
  • その覚悟があると対等な交渉ができる
  • 実際に失うことは稀だが心構えは重要
コツ④断られても態度を変えない
  • 交渉が通らなくても普通に仕事を続ける
  • 次の機会に再度交渉
  • 態度が変わると信頼を失う
5-6. 新規案件での最初から高単価を狙う
既存クライアントへの交渉だけでなく最初から高単価の案件を狙う方法もあります。
方法
  • ①高単価案件に絞って応募する：文字単価1.5円以上の案件を中心に
  • ②応募文で価値をアピール：専門性実績を具体的に / この人は単価に見合うと思わせる
  • ③自分から単価を提示する：希望単価〇円と明記 / 最初からこの単価で受ける人と認識される
第6章：「単価アップ」のマインドセット
テクニックだけでなく、考え方も重要です。単価アップを実現するマインドセットを身につけましょう。
6-1. 「安いから」で選ばれる人にならない
「安いから」選ばれる状態は、とても不安定です。
【「安さ」で選ばれる問題点】
  • もっと安い人が現れたら、切られる
  • 「いつでも替えが効く」と思われる
  • 単価を上げると、離れていく
  • 自己肯定感が下がる
【「価値」で選ばれる状態】
  • 「〇〇さんだから頼みたい」と言われる
  • 少々高くても、選ばれる
  • クライアントが「失いたくない」と思う
  • 単価アップも受け入れられやすい
目指すのは「安いから選ばれる」ではなく「価値があるから選ばれる」状態です。
6-2. 「時給」で考える習慣をつける
文字単価だけでなく、「時給換算」で考える習慣をつけましょう。
【時給換算の例】
案件A：文字単価1円×3,000文字=3,000円 → 執筆時間3時間 → 時給1,000円
案件B：文字単価0.8円×3,000文字=2,400円 → 執筆時間1.5時間 → 時給1,600円
→ 文字単価が低くても、時給が高い場合がある
→ 逆もある（高単価でも、時間がかかりすぎると時給は低い）
【時給を上げる方法】
  • 単価を上げる
  • 執筆スピードを上げる
  • リサーチ時間を短縮する
  • 得意なジャンルに絞る
  • 効率の悪い案件を断る
6-3. 「断る勇気」を持つ
単価アップのためには、「断る勇気」も必要です。
【断るべき案件】
  • 明らかに割に合わない低単価
  • 修正が多すぎるクライアント
  • コミュニケーションがストレスになるクライアント
  • 自分の成長につながらない案件
〇〇様、お声がけいただきありがとうございます。大変恐れ入りますが、現在の稼働状況と単価の条件を踏まえ、今回はお受けすることが難しい状況です。また条件が合うタイミングがございましたら、ぜひご連絡いただければ幸いです。よろしくお願いいたします。
断ることで時間が空き、その時間でより良い案件を探せます。
6-4. 「長期視点」で考える
単価アップは、一朝一夕には実現しません。長期視点で考えましょう。
1
【短期】1〜3ヶ月
  • 実績を作る
  • 継続案件を獲得する
  • 基礎スキルを身につける
2
【中期】3〜6ヶ月
  • 単価を少しずつ上げる
  • 得意ジャンルを深める
  • 高単価案件に挑戦
3
【長期】6ヶ月〜1年
  • 文字単価2円以上を安定させる
  • 専門家として認識される
  • 指名で依頼が来る

まとめ：「いくらで受けるか」はあなたが決める
この章では、単価が上がる人と上がらない人の分岐点を学びました。
単価が上がらない理由
  • 「実績を積めば自然と上がる」は幻想
  • 低単価の仕事を「数」でカバーしようとする
  • 「まだ自分には早い」と思って挑戦しない
  • 単価交渉を避ける
単価が上がる人の共通点
  • 高単価案件に「挑戦」している
  • 「価値」を提供している
  • 単価交渉をしている
  • 「専門性」を磨いている
  • 「断る」ことができる
単価を上げる7つの具体策
  • ① より高単価の案件に応募する
  • ② 既存クライアントに単価交渉する
  • ③ 得意ジャンルを作る
  • ④ 構成作成から受注する
  • ⑤ 直接契約を目指す
  • ⑥ スキルを増やす
  • ⑦ 実績を「見える化」する
単価は「あなたがいくらで受けるか」で決まります。待っていても単価は上がりません。自分から行動することで、単価は上がります。

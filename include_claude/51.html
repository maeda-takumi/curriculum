第5章：価格設定と価値の伝え方【30分】

5-1. 価格設定の3つのアプローチ

アプローチ①：市場相場から決める → noteで同ジャンルの有料記事の価格を10個調べ、最初は「平均値の80%」程度で設定。

アプローチ②：「得られる成果」から逆算する → 読者が得られる「経済的な成果」から逆算（月5万円×12ヶ月=60万円の成果なら2,980円は安い）。

アプローチ③：段階的に値上げする → 最初は低価格（500〜980円）→ レビューが付いたら値上げ → 内容アップデート時に再度値上げ。

5-2. 「価値」を伝えるセールスライティング

商品ページの「型」（販売ページの構成）

①キャッチコピー → ②問題提起（読者の痛みを言語化）→ ③共感（「自分も同じだった」）→ ④解決策の提示 → ⑤得られる成果（具体的な数字で）→ ⑥商品の詳細（目次・内容一覧）→ ⑦お客様の声（あれば）→ ⑧価格の提示 → ⑨購入ボタン → ⑩よくある質問

💡 セールスライティングの禁止事項：嘘の実績や数字を書かない / 過度な煽り表現を使わない / 「誰でも」「絶対」「100%」は避ける / 他人の商品を貶めない。正直に、具体的に、読者目線で書くのが長期的に最も売れる方法です。

第7章：商品化のメンタルブロックを外す【20分】

メンタルブロック①〜⑦

①「自分に商品を作る資格がない」→ 月5万円稼いだ経験は月0円の人にとって「最高の教科書」。「3ヶ月前の自分」に教えてあげる気持ちで作る。

②「批判されたらどうしよう」→ 批判する人はあなたのターゲットではありません。最初は匿名（ペンネーム）で出しても構いません。

③「売れなかったら恥ずかしい」→ 重要なのは「売れたかどうか」ではなく「出したかどうか」。出せば改善できる。

④「完璧じゃないから出せない」→ 80%の完成度で出して購入者のフィードバックで改善する方が圧倒的に速い。「随時アップデートしていきます」と書けばOK。

⑤「お金をもらうのが申し訳ない」→ あなたの知識は試行錯誤に何十時間もかけた結果。適正な価格は双方にとってプラス。

⑥「もう同じ商品がたくさんある」→ 競合がいる＝需要がある証拠。あなたの体験と切り口は唯一無二。

⑦「時間がない」→ フェーズ4で21時間の空き時間を作りました。1日30分×14日＝7時間でも3,000文字の商品は作れます。

【ターゲット】副業を始めたいが何から始めればいいかわからない会社員 / 【私の強み】Claude活用、提案文の書き方、効率化の知識 / 【価格帯】980〜2,980円

【各提案に含めてほしい情報】1.商品タイトル案 / 2.ターゲットの具体的な悩み / 3.商品で得られる成果 / 4.大まかな章構成（5〜7章）/ 5.差別化ポイント

【プロンプト②：商品の章構成を詳細化する】

以下のnote有料記事の構成を詳細化してください。【タイトル】〇〇 / 【全体の文字数】8,000〜10,000文字。各章について：1.伝えるべきポイント3つ / 2.読者が知りたいことTOP3 / 3.入れるべき具体例やデータ / 4.推奨文字数

【プロンプト③：章ごとの下書きを生成する】

以下の章について、note有料記事の本文を書いてください。【記事テーマ】〇〇 / 【この章のタイトル】〇〇 / 【文字数】1,500〜2,000文字。【トーン】初心者に寄り添う、親しみやすい文体 / 「先輩が後輩に教える」イメージ。【禁止】「〜と言えるでしょう」「いかがでしたか」「架空の体験談や統計データは作らない」

【プロンプト④：販売ページのコピーを生成する】

以下のnote有料記事の「販売ページ」を作成してください。【タイトル】〇〇 / 【ターゲット】〇〇 / 【価格】〇〇円 / 【著者の実績】〇〇。含める要素：1.キャッチコピー / 2.ターゲットの悩み（3〜5個）/ 3.共感ストーリー（100〜150文字）/ 4.得られること（3つ）/ 5.商品内容の紹介 / 6.特典の紹介 / 7.こんな方におすすめ / 8.著者プロフィール / 9.最後の一言。文字数：800〜1,200文字 / トーン：正直で親しみやすい。煽り表現は使わない。

【プロンプト⑤：タイトル案を量産する】

以下のnote有料記事のタイトル案を10パターン作成してください。【条件】30〜50文字 / 具体的な数字を入れる / ターゲットが「自分のことだ」と思えるものにする。①成果＋期間＋方法 / ②ターゲット＋悩み＋解決策 / ③ストーリー＋数字 の3パターンを混ぜる。【禁止】「最強」「神」「知らないと損」「衝撃」等の煽り表現

4-6. 時間単価を上げるための仕組み設計

フェーズ4｜再現性・自動化・仕組み化編

このマニュアルを読み終える頃には

「時給1,500円→3,000円」への具体的な道筋が見えています

はじめに：なぜ「時間単価」にこだわるのか

4-5までで、効率的に案件をこなす仕組みが完成しました。4-6では、その仕組みを活かして「時間単価」を上げていきます。

【時間単価が収入を決める理由】

月収 ＝ 時間単価 × 稼働時間

・時給1,500円 × 20時間 ＝ 月収30,000円 / ・時給2,500円 × 20時間 ＝ 月収50,000円 / ・時給4,000円 × 20時間 ＝ 月収80,000円

副業は「稼働時間に限界」があります。本業・家事・育児があるので月20〜40時間が現実的な上限。だから「時間単価を上げる」ことこそが収入アップの鍵なのです。

【シナリオ：「月5万円の壁」を突破したい】

月の売上：約65,000円 / 稼働時間：約38時間/月 / 案件内訳：A様（美容）月5本×7,000円＝35,000円 / B様（転職）月3本×6,000円＝18,000円 / C様（副業）月2本×6,000円＝12,000円

計算すると：65,000円 ÷ 38時間 ＝ 時給1,710円。これ以上案件を増やすのは時間的に厳しい。月10万円を目指すには「時間単価を上げるしかない」と気づいた。

第1章：時間単価の現状を把握する【所要時間30分】

💰 この章で得られる収益効果：自分の「本当の時給」がわかる / どこを改善すれば単価が上がるか明確になる / 目標設定が具体的にできるようになる

1-1. 「思っている時給」と「本当の時給」は違う

「1記事7,000円で、2時間半で書けるから、時給2,800円だ！」という勘違い。

実際の作業時間を分解すると：①案件獲得（営業）35分 / ②ヒアリング・準備65分 / ③執筆110分 / ④納品・修正45分 / ⑤事務処理5分 = 合計260分＝4時間20分

本当の時給：7,000円 ÷ 4.33時間 ＝ 1,617円。「2時間半で時給2,800円」のつもりが「4時間20分で時給1,617円」だった！

原因1：新規案件の営業時間 → リピート率60%以上に 効果：+15%

原因5：単価の低いクライアントに依存 → クライアントポートフォリオを分散 効果：+20%

第6章：外注化で「時間を買う」【所要時間30分】

💰 この章で得られる収益効果：自分の時間を「高単価な仕事」に集中 / 受注できる案件数が2〜3倍に / 「仕組みで稼ぐ」へシフト

6-1. 外注化で「時給」が上がる仕組み

外注なし：案件単価10,000円 / 作業時間4時間 / 時給2,500円

外注あり：案件単価10,000円 / 外注費5,000円 / 自分の作業（ディレクション）1時間 / 利益5,000円 / 時給5,000円 → 時給が2倍に！

空いた3時間で：さらに案件を受注 / 高単価案件の営業 / 専門性を高める勉強 → さらに収入アップ

6-2. 外注化する「タイミング」

まだ早いケース：月の案件数が5件以下 / 自分の時給が2,000円以下 / 品質チェックができるレベルにない

始めてもいいケース：□ 月の案件数が10件以上 / □ これ以上増やすのは時間的に限界 / □ 記事の品質チェックができる

シナリオの場合：月10件・時間的に限界・品質チェックできる → 外注化を検討してもいい段階

1-2. 教えることで「自分のスキル」が上がるメカニズム

「プロテジェ効果」と呼ばれる現象があります。人に教えることで、自分の理解が深まるという効果です。曖昧なまま「なんとなくできていた」ことが、人に説明しようとすると「実は理解できていなかった」と気づかされる。逆に、説明できるようになったとき、そのスキルは完全に自分のものになります。

1-3. 「少し先を歩く先輩」が最強の教師

初心者にとって最も役立つ教師は「完璧な専門家」ではなく「少し先を歩く先輩」です。悩みが生々しく、解決策が具体的で、「自分にもできそう」な再現性がある。フェーズ5まで到達したあなたは、フェーズ0〜4にいる人の「完璧な先輩」です。

第4章：価格設定と受講者獲得〔40分〕

4-1. 教育ビジネスの価格設定の原則

原則1：得られる「成果の価値」から逆算する（「月収10万円稼げるようになる」スクール → 1年で120万円の価値 → 10〜30万円のスクール費用は妥当）

原則2：「価格＝価値の宣言」という発想を持つ（安すぎると「効果があるか不安」と感じられる）

原則3：段階的に価格を上げる（最初は低く設定 → 実績が増えたら値上げ → 満席にする → さらに値上げ）

4-2. 受講者獲得の具体的な方法

方法1：SNSからの集客 → note記事やSNS投稿で「スクールの内容の一部」を無料公開 → LINE登録 → スクールへの誘導

方法2：説明会・無料セミナーの開催 → Zoomで1〜2時間の無料セミナー → 内容の一部を提供 → 参加者向けにスクールを案内

方法3：既存受講者からの紹介 → 「友人紹介割引」を設定 → 満足した受講者が口コミで紹介

方法4：コーチング受講者のスクールへの移行 → 個別コーチングを受けた人が「グループでも学びたい」となるのは自然な流れ

9-2. 「教えることで学ぶ」サイクルを回す

受講者の質問への回答 → 自分の知識の整理 → 回答をコンテンツ化 → SNS投稿・記事化 → 新しい受講者の獲得 → 新しい質問。このサイクルを意識的に作ることで、教えるほど知識が深まり、コンテンツが増え、信用が積み上がります。

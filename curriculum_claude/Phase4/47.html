3-1. リサーチ時短プロンプト【4種類】

【プロンプト①：キーワードの基礎知識を一気に把握】

用途：新しいキーワードの記事を書く前に基礎知識を素早くインプット / 時短効果：25分→5分（20分削減）

「〇〇」について記事を書きます。以下を教えてください：1.〇〇とは何か（100文字）/ 2.メリット5つ（各30文字）/ 3.デメリット3つ / 4.よくある疑問5つ / 5.関連キーワード10個 / 6.検索する人の悩みTOP3 / 7.最新トレンド（2024年）

【プロンプト②：競合記事の見出し分析】

用途：検索上位記事の見出し構成を分析 / 時短効果：20分→3分（17分削減）

以下は「〇〇」で検索上位3記事の見出しです。分析して教えてください：1.3記事に共通する見出し / 2.差別化ポイント / 3.足りない視点 / 4.読者が本当に知りたいこと / 5.上位表示に必要な要素。【記事1〜3の見出し】（貼り付け）

【プロンプト③：専門用語の簡単解説】

時短効果：10分→1分。条件：各50文字以内 / 難しい言葉は使わない / 具体例を1つ入れる。【用語リスト】1.〇〇 / 2.△△ / 3.□□

【プロンプト④：ペルソナの深掘り】

時短効果：15分→3分。教えてほしいこと：1.検索する理由 / 2.悩みTOP5 / 3.知りたいことTOP5 / 4.読んだ後に取りたい行動 / 5.躊躇する理由TOP3 / 6.背中を押すために必要な情報

3-2. 構成案作成プロンプト【3種類】

【プロンプト⑤：SEO記事の構成案を一発生成】

時短効果：20分→5分。条件：タイトル案3パターン / リード文の方向性 / H2見出し（5〜7個）/ 各H3見出し（2〜3個）/ 各見出しで書く内容のポイント / まとめの方向性 / 注意：読者の悩みに寄り添う構成

【プロンプト⑥：構成案のブラッシュアップ】

時短効果：10分→2分。チェック観点：読者の疑問に答えられているか / 見出しの順番 / 抜けている内容 / 削除すべき内容 / SEO的な改善点 / 差別化ポイント

【プロンプト⑦：見出しのキャッチコピー化】

時短効果：10分→2分。条件：キーワードは残す / 読者のベネフィットを入れる / 具体的な数字を入れる / 30文字以内

3-3. 本文執筆プロンプト【5種類】

【プロンプト⑧：リード文（導入文）を生成】

【プロンプト⑨：見出しごとに本文を生成】

時短効果：15分→7分（8分×6見出し=48分削減）。禁止：「〜と言えるでしょう」「いかがでしたか」「同じ文末を3回続けない」「また・さらにの多用」

【プロンプト⑩：まとめ（結論）を生成】

【プロンプト⑪：具体例・体験談を生成】

時短効果：10分→2分。条件：リアリティのある内容 / 数字を入れる（〇〇円、〇〇%）/ ビフォーアフターを入れる / 「自分もこうなれる」と思える内容

【プロンプト⑫：箇条書きを文章化】

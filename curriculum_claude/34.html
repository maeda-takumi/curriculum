AI副業スクール

3-4. 文章×画像×動画をつなぐ"設計思考"

フェーズ3｜ライティングを『武器』にする応用編

はじめに：「書ける人」から「設計できる人」へ

ここまでのフェーズ3で、あなたは多くのスキルを身につけました。セールスライティングの基本構造、LPの設計、SNS・動画台本・広告文への横展開…。一つ一つが、AI副業で稼ぐための「武器」です。

しかし、ここで一度立ち止まって考えてみてください。あなたが提供しているのは、本当に「文章」だけでしょうか？

クライアントが本当に欲しいのは、「文章」ではありません。「売上」であり、「集客」であり、「ブランド構築」です。文章は、その目的を達成するための「手段の一つ」に過ぎません。

ここに、多くのライターが気づいていない「大きなチャンス」があります。クライアントは本来、文章・画像・デザイン・動画・SNS投稿・広告、それらすべてが「一貫した戦略のもとに設計されている状態」を望んでいます。しかし現実には、文章はAさんに、デザインはBさんに、動画はCさんにとバラバラに外注している。

【設計思考とは】

設計思考とは、「文章」「画像」「動画」をバラバラに考えるのではなく、クライアントの目的達成のために「統合的に設計する」思考法です。LPの文章だけでなく、デザインの方向性も提案する。SNS投稿の文章だけでなく、画像の構成も考える。動画台本だけでなく、サムネイルやタイトルとの連携も設計する。「書く人」から「設計する人」へ。この進化が、あなたの市場価値を大きく高めます。

⚠ 注意：この章で扱うのは、あくまで「ライティング（言語設計）」の視点から、画像・動画・SNSとの連携を設計するノウハウです。YouTubeチャンネルの伸ばし方、SNSのフォロワーの増やし方、動画編集テクニックなど「各プラットフォーム固有の運用ノウハウ」は本スクールの範囲外となります。

【なぜ設計思考が重要なのか】

  • ① 単価が上がる：「文章だけ」より「文章＋画像構成案」の方が高単価。設計を含めると単価は2〜3倍に
  • ② クライアントの満足度が上がる：一貫した設計の方が成果が出る。「この人に任せれば安心」と思われる
  • ③ 継続案件・紹介が増える：「設計できる人」は希少。一度信頼されると長期的なパートナーになれる
  • ④ 自分のビジネスにも活かせる：独立・起業の土台になる

【この章の目標】

  • 文章・画像・動画を「統合的に設計する」視点を身につける
  • クライアントに「設計案」として提案できるようになる
  • 「書く人」から「設計する人」へと進化する
  • 設計込みの案件を獲得し、単価を上げる
  • Claudeを活用して設計業務を効率化する方法を学ぶ

第1章：設計思考の基本

1-1. 「作業者」と「設計者」の違い

【作業者の仕事】

● 指示された文章を書く / 指定されたフォーマットで納品する / 言われたことをやる / 修正依頼があれば対応する

作業者の特徴：単価が上がりにくい / 替えがきく / 価格競争に巻き込まれやすい

【設計者の仕事】

● クライアントの目的を理解する / 目的達成のための全体設計を考える / 文章だけでなく、画像の方向性や動画構成の「言語設計」まで提案する / 成果にコミットする / 改善提案まで行う

設計者の特徴：単価が高い / 替えがきかない（この人でないとダメ）/ 継続的な仕事が生まれやすい / 価値で選ばれる

重要なのは、設計者になるために「デザインの技術」や「動画編集の技術」は必要ないということです。設計者に求められるのは、技術ではなく「思考」と「言語化力」です。

1-2. 設計思考の3つの視点

【視点①：目的から逆算する】

作業者は「LPを書いてください」→「わかりました、書きます」。設計者は「このLPの目的は何ですか？」「ターゲットは誰ですか？」「どこからの流入を想定していますか？」「LPの前後の導線はどうなっていますか？」と確認してから設計を始める。

【視点②：全体を俯瞰する】

「LP」が全体の中でどう位置づけられるかを見る。広告→LP→購入の流れ、SNS→LP→メルマガ登録の流れ。LP単体ではなく、前後の導線も含めて考える。

【視点③：要素の連携を考える】

文章とデザインのトーンは一貫しているか / SNSの投稿文と画像上のテキストはマッチしているか / 動画の台本とサムネイル上の文言は連動しているか / 広告のコピーとLP冒頭のメッセージは統一されているか。「一貫性を言葉で設計する」ことがライターにしかできない設計思考の核心です。

1-3. 設計者に求められるスキル

【設計者に必要な5つのスキル】

1. ヒアリング力：クライアントの目的、課題を引き出す / 言語化されていないニーズを把握する

2. 構造化力：情報を整理し、全体像を描く / 目的→戦略→施策の流れを設計する

3. 言語化力：設計意図を言葉で説明できる / デザイナーや動画編集者に意図を正確に伝える指示書を書ける

4. 連携力：文章・画像・動画を「言葉のトーン」で統一する / 各メディアごとの文章の書き分け

5. 改善力：成果を踏まえて文章の改善案を出す / 仮説を立てて検証するプロセスを実行する

これらのスキルはすべて「言語」がベースになっています。つまり、ライティングスキルを持っている人は、設計者になるための素地をすでに持っているのです。

第2章：文章と画像の連携設計

2-1. LPにおける文章と画像の関係

【LPでの役割分担】

【文章の役割】論理的な説明 / 詳細な情報提供 / 感情に訴えるストーリー / 行動を促すCTA

【画像・デザインの役割】第一印象を決める / 読みやすさを左右する / 感情的なインパクト / 信頼感の演出

【連携のポイント（ライターが設計するべき部分）】文章とデザインのトーンを「言葉で」指定する / デザイナーに伝わる「言語化された画像指示」を作成する

人間がLPにアクセスしたとき、最初に目に入るのは「画像」であり、次に「見出し（大きな文字）」、最後に「本文」です。この「視覚→文字」の順番を理解した上で、両者の「言葉のトーン」を設計するのが設計思考です。

2-2. トーン＆マナーを「言語で」統一する

【トーン＆マナー設計の具体例】

【例①：高級感を出したいLP】

文章のトーン：フォーマルな敬語、専門用語を適度に使用 / 例：「厳選された素材を、匠の技で仕上げました」

画像への指示：モノトーン基調、余白多め、洗練されたセリフ体フォント / 色味：黒、金、白、ダークネイビー

【例②：親しみやすさを出したいLP】

文章のトーン：カジュアルな口語体、語りかけるような文体 / 例：「え、まだ知らないの？これ、マジでいいよ！」

画像への指示：明るい色、丸みのあるイラスト、手書き風フォント / 色味：パステルカラー、暖色系

【例③：信頼感・実績を強調したいLP】

文章のトーン：データを多用、数値で説得 / 例：「導入企業3,200社。顧客満足度97.3%」

画像への指示：グラフ、チャート、企業ロゴの並び / 色味：青系、白、グレー

【NGパターン】

文章：カジュアルで親しみやすい + 画像：高級感のあるモノトーン → チグハグな印象、信頼感が下がる

トーン＆マナーを統一するコツは、最初に「このLP全体のペルソナ」を言葉で定義することです。「もしこのLPが人だったら、どんな人物か？」この「ペルソナ定義文」をクライアントやデザイナーと共有すれば、全員が同じ方向を向いて制作できます。

2-3. ライターが提案できる「画像指示書」

【ライターが提案できること】

  • 画像の配置場所（どの文章の横に画像を入れるか）
  • 画像の内容（何を見せるべきか ─ 言葉で説明）
  • 画像のトーン（明るい/落ち着いた/高級感など ─ 言葉で指定）
  • テキストを載せる場合の文言（コピーライティングの領域）
  • 参考イメージのURL（競合やPinterestから）
  • 人物写真のイメージ（性別、年齢層、表情、服装 ─ 言葉で描写）

「画像はデザイナーさんにお任せします」ではなく、「こんな画像が効果的だと思います。指示書を作りました」と提案する。この一言の違いが、「作業者」と「設計者」の違いを作ります。

【LP画像指示書の例】

【ファーストビュー】・メイン画像：30代男性がパソコンに向かって笑顔で作業している写真 ・キャッチコピーをオーバーレイ：「未経験から3ヶ月で月10万」 ・文字色は白、背景は暗めのトーン ・CTAボタン：オレンジ系、テキスト「無料で始める」

【問題提起セクション】・イメージ：頭を抱える人のシルエット ・背景を薄いグレーにして重苦しさを演出 ・「こんなお悩みありませんか？」の吹き出しデザイン

【解決策セクション】・明るいトーンに切り替え（問題提起セクションとの対比）・「3つの特徴」をアイコン＋短文で表示（アイコンは丸型、青系）

【お客様の声セクション】・人物写真＋星評価のアイコン ・引用符（"）デザインで信頼感を演出

【料金セクション】・カード型デザインで3プランを横並び ・おすすめプランに「BEST」バッジ ・特典はチェックマークアイコン付き

2-4. SNS投稿における文章と画像の連携

⚠ 注意：ここで解説するのは、SNS投稿における「テキスト（キャプション・画像上の文言）をどう書くか」というライティング視点のノウハウです。SNSのフォロワーを増やすアルゴリズム攻略や、リーチを伸ばすための運用ノウハウは本講座の範囲外です。

【Instagramカルーセル投稿の「テキスト設計」】

【1枚目（表紙）】タイトルは15文字以内で「フック」を入れる。例：「知らなきゃ損！副業で月10万稼ぐ5つのコツ」 / 背景色とのコントラストを確保する旨をデザイナーに指示

【2〜9枚目（本文）】1スライド1メッセージ（30〜50文字）/ ナンバリング（①②③）で構造化 / テキストの配置は中央揃え、余白は十分に取る旨を指示

【10枚目（CTA）】行動を促す一言：「保存してね」「フォローしてね」/ プロフィールリンクへの誘導文を含める

2-5. Claudeで画像指示書を効率的に作成する

【プロンプト例：LP画像指示書の生成】

あなたはWebマーケティングのプロのライターです。以下のLPについて、デザイナーに渡す「画像指示書」を作成してください。

【LP概要】・商品：副業ライター養成講座 ・ターゲット：30代会社員、副業を始めたい ・トーン：信頼感がありつつも親しみやすい

【出力形式】各セクションごとに：画像の内容（言葉で具体的に）/ トーン・色味の指定 / 画像上に載せるテキスト（コピー）/ テキストの配置やサイズの方向性 / デザイナーへの注意事項

【プロンプト例：Instagram画像テキスト指示書】

以下のInstagramカルーセル投稿について、各スライドの「画像上テキスト」と「キャプション」を作成してください。

【投稿内容】テーマ：副業ライターが最初の1万円を稼ぐロードマップ / 形式：カルーセル10枚

【出力形式】各スライドごとに：画像上テキスト（30字以内）/ 補足テキスト（15字以内）/ トーン・色味の指定 / 最後に：キャプション（400文字以内）/ ハッシュタグ10個

第3章：文章と動画の連携設計 ─ ライターの領域

⚠ 注意：ここで解説するのは、動画における「台本の書き方」「タイトルの付け方」「サムネイル上のテキスト設計」など、あくまでライティング（言語設計）の観点からのノウハウです。撮影技術、動画編集テクニック、YouTubeチャンネルを伸ばすためのアルゴリズム攻略などは本講座の範囲外となります。

3-1. YouTube動画の「言語設計」要素

【ライターが担当できる動画の言語設計要素】

【台本】話す内容の構成と文章 / 冒頭15秒のフック / 途中の「引き」 / CTA文言 / テロップ用のキーフレーズ

【タイトル】検索キーワードを含んだ魅力的なタイトル / クリックを促す言葉選び / サムネイルとの「補完関係」を意識したタイトル設計

【サムネイル上のテキスト】一瞬で内容が伝わる短いコピー（6〜12文字）/ 感情を動かす言葉選び / 文字の配置・サイズの方向性をデザイナーに指示

【概要欄の文章】動画の要約文 / 関連リンクの整理 / LP・SNSへの誘導テキスト

3-2. サムネイルテキスト・タイトル・台本の一貫性

【良い例：言葉のメッセージが一貫している場合】

サムネイルテキスト：「月10万」（大きく太文字）/ タイトル：「副業で月10万稼ぐ方法｜初心者向け完全解説」/ 台本冒頭：「今日は副業で月10万稼ぐ具体的な方法を解説します」→ すべてが「月10万」というコアメッセージで統一

【悪い例：メッセージがバラバラの場合】

サムネイルテキスト：「副業で稼ぐ」（漠然）/ タイトル：「ライター講座のご紹介」/ 台本冒頭：「今日は私の経歴を紹介します」→ 視聴者が「思ってたのと違う」と感じて即離脱

【ライターとして意識すべきこと】

サムネイルテキストとタイトルは「補完関係」にする。サムネイル＋タイトルで1つの完全なメッセージを形成する。例：サムネイル「月10万」＋タイトル「副業ライター初心者が稼ぐ完全ロードマップ」

3-3. 「コアメッセージ先行」の同時設計フレームワーク

【同時設計の4ステップ】

【STEP1】コアメッセージを1文で決める：「この動画を一言で言うと？」例：「副業ライターが最短で月10万稼ぐ方法」

【STEP2】コアメッセージからサムネイルテキストを設計：大きく見せるテキスト「月10万」/ 補足テキスト「副業ライター」/ デザイナーへの指示：驚き表情の人物写真、黄色×黒の配色

【STEP3】コアメッセージからタイトルを設計：キーワード：副業、ライター、月10万、稼ぐ / タイトル案：「【副業】ライター初心者が月10万稼ぐ完全ロードマップ」

【STEP4】コアメッセージから台本の骨子を設計：冒頭フック「副業で月10万稼ぎたい人、この動画を見れば具体的なステップが全部わかります」/ 本編構成：月10万達成のステップを5つに分けて解説

ライターが台本だけでなくタイトルやサムネイルテキストまで提案すると、クライアントは「台本だけ頼んだのに、ここまで考えてくれた」と感動します。追加で提案しているのは「テキスト」であって、画像制作ではないため、ライティングスキルの延長で自然にできることです。

第4章：LP・SNS・動画を「言語」で統合設計する

4-1. コアメッセージを軸にした統合設計

【メディアごとの「書き分け」設計例】

【コアメッセージ】「未経験から3ヶ月で月10万稼げる副業ライター講座」

【LP】5,000〜10,000文字の説得力ある長文 / ベネフィット、お客様の声、料金を詳述 / CTA：「今すぐ申し込む」/ 画像指示書を添付 / トーン：信頼感＋期待感

【YouTube台本】3,000〜5,000文字（10〜15分動画）/ 「月10万稼ぐ方法」を一部無料公開 / サムネイルテキスト＋タイトル案を添付 / トーン：親近感＋専門性

【Instagram投稿文】キャプション100〜300文字 / 画像上テキスト30文字以内×10枚 / 「月10万稼ぐコツ」を小分けに発信 / トーン：カジュアル＋共感

【広告文】30〜100文字の短文勝負 / 「月10万」を軸にした訴求 / 複数バリエーション（A/Bテスト用）/ トーン：具体的＋緊急性

4-2. 導線設計を「テキスト」で可視化する

【導線設計テキストの例】

①YouTube概要欄：「この動画で紹介した講座の詳細はこちら → [LP URL] 今なら無料体験レッスンを実施中です」

②Instagramプロフィール：「未経験→月10万のロードマップ公開中👇」

③Instagram投稿のCTA：「もっと詳しく知りたい方はプロフィールのリンクから！」

④広告文のCTA：「今すぐ無料体験を予約 →」

⑤LP冒頭ファーストビュー：各流入元との一貫性を意識（YouTube経由：「動画でお伝えした内容をさらに詳しく」/ 広告経由：「広告で見たあの講座の全貌がわかります」）

4-3. 統合設計の提案書フォーマット

【クライアントへの統合設計提案書】

  • 目的：〇〇講座の集客・販売 / 月間申込数を10件→30件に増加
  • コアメッセージ：「未経験から3ヶ月で月10万稼げる」
  • 施策設計（ライティング納品物）：1.LP原稿＋画像指示書（セールスライティング原稿5,000文字＋各セクションの画像指示書）/ 2.YouTube動画×3本分のライティングセット（台本＋タイトル案＋サムネイルテキスト案＋概要欄テキスト）/ 3.Instagram投稿×10本のライティングセット / 4.広告文（リスティング×3セット＋SNS広告×3セット）
  • スケジュール：第1週：ヒアリング、コアメッセージ＆全体設計 / 第2週：LP原稿・画像指示書 / 第3週：YouTube台本・SNS原稿 / 第4週：広告文作成、全体確認、納品

第5章：クライアントへの提案方法

5-1. 提案の基本姿勢

【設計者の対応】

クライアント「LP書いてください」→ 設計者：「このLPの目的は何ですか？」「ターゲットはどんな方ですか？」「LPと連動したSNSの投稿文や広告文も一緒に設計しましょうか？テキストのトーンを統一した方が成果が出やすいので」→ 成果を出すための情報を引き出し、「ライティングの範囲を広げる」提案をする

5-2. ヒアリングチェックリスト

【必ず確認すること】

  • □ 目的（売上UP？リード獲得？ブランディング？）
  • □ ターゲット（誰に届けたいか）
  • □ 商品・サービスの強み
  • □ 競合との違い
  • □ 流入経路（どこから人が来るか）

まとめ：設計思考の第一歩

【この章で学んだこと】

  • 「作業者」と「設計者」の根本的な違い
  • 設計思考の3つの視点（目的逆算、全体俯瞰、要素連携）
  • 設計者に必要な5つのスキル（すべて言語力がベース）
  • トーン＆マナーを「言語で」定義し統一する方法
  • ライターが作成できる「画像指示書」の具体例
  • SNS投稿における「テキスト設計」の方法
  • 動画の「言語設計」要素（台本・タイトル・サムネイルテキスト）
  • コアメッセージ先行の同時設計フレームワーク
  • 複数メディアを「テキストの一貫性」で統合設計する方法
  • クライアントへの提案方法（ヒアリング、提案書、トーク）
  • 設計者としての単価設定と見積もりの作り方
  • Claudeを使った設計ライティングの効率化プロンプト集

【設計思考の本質】

設計思考とは、「クライアントの目的達成のために、すべてのテキストを統合的に設計する」という考え方です。文章、画像上のテキスト、動画の台本、広告文、CTAの文言、導線上の誘導文…すべての「言葉」に一貫性を持たせること。これはライターだからこそできる仕事です。

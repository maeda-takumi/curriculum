AI副業スクール

3-3. SNS・動画台本・広告文への横展開

フェーズ3｜ライティングを『武器』にする応用編

はじめに：セールスライティングは「万能スキル」

前章までで、セールスライティングの基本構造とLP作成の方法を学びました。セールスライティングは、LPだけのスキルではありません。あなたが学んだ「人を動かす言葉の設計技術」は、あらゆる媒体に応用できます。

SNS投稿、YouTube台本、TikTok動画、Web広告、メルマガ、LINE配信…。これらに、セールスライティングの原則が活きてきます。

【なぜ横展開が重要なのか】

  • ① 収益の柱が増える：LPだけでなく、SNS運用、動画台本、広告運用など、複数のスキルを持つことで収入源が多様化
  • ② クライアントへの提案幅が広がる：「LPだけ」より「LP＋SNS＋広告」の方が単価が高い
  • ③ 市場価値が上がる：複数媒体を横断して設計できる人は希少
  • ④ 自分のビジネスにも活かせる：自分の商品・サービスを売る力がつく

【この章の目標】

  • セールスライティングを各媒体に応用する方法を理解する
  • SNS投稿、動画台本、広告文の「型」を習得する
  • Claudeを使って各媒体のコンテンツを効率的に作成できる
  • 「LP→SNS→広告→動画」の一貫した設計ができるようになる

第1章：媒体が変わっても変わらない原則

1-1. セールスライティングの普遍原則

【普遍原則①：3つのNOT】

どの媒体でも、読者・視聴者は NOT READ（読まない・見ない）/ NOT BELIEVE（信じない）/ NOT ACT（行動しない）という3つの壁があります。媒体が変わっても、この課題は同じです。

【普遍原則②：ターゲット起点】

誰に向けて発信するか / その人はどんな悩みを持っているか / どんな言葉に反応するか。ターゲットを明確にすることは、どの媒体でも最も重要なステップです。

【普遍原則③：ベネフィット訴求】

フィーチャー（機能）ではなくベネフィット（価値）。「何ができるか」ではなく「どう変われるか」。これは、すべての媒体に共通する鉄則です。

【普遍原則④：感情→論理】

まず感情に訴えかけ、その後、論理で裏付ける。感情で「欲しい」、論理で「買っていい理由」。この順番は、どの媒体でも同じです。

1-2. 媒体ごとに変わるもの

① 文字数・尺（Twitter：140文字 / Instagram：冒頭3行が勝負 / YouTube：5〜20分 / TikTok：15秒〜3分）

② フォーマット（SNS：テキスト、画像、動画 / YouTube：動画＋サムネ＋タイトル）

③ ユーザーの心理状態（SNS：暇つぶし、情報収集 / YouTube：学びたい / 広告：基本的に見たくない）

④ アルゴリズム（各プラットフォームの表示ロジック）

1-3. 「型」を持つことの重要性

【型を持つメリット】

  • ① 迷わない：何を書くか、型に沿えばOK
  • ② 速い：ゼロから考える必要がない。Claudeにも指示しやすい
  • ③ 安定する：型に沿えば、一定の質を保てる
  • ④ 改善しやすい：型があるから、どこを改善すべきかわかる

第2章：Twitter（X）への横展開

2-1. Twitterの特徴

日本語で140文字。スクロールしながら、一瞬で判断。1投稿に費やす時間は1〜2秒。いいね、RT、リプライが多いと拡散。

2-2. Twitterでの3つのNOT

【NOT READ対策】1行目で「自分ごと」と思わせる / 数字を使う / 「○○な人へ」と呼びかける

【NOT BELIEVE対策】具体的な数字を入れる / 自分の経験談を語る / 実績を示す

【NOT ACT対策】明確なCTAを入れる：「詳しくはプロフへ」「リプで質問どうぞ」

2-3. Twitterの型（テンプレート）

【型①：問題提起→解決策型】

「英語が話せない」と悩んでいませんか？実は、日本人が英語を話せない原因は「勉強法」ではなく「○○」なんです。この○○を変えるだけで、3ヶ月で会話できるようになります。詳しくはプロフのリンクから👇

【型②：実績・ノウハウ型】

副業ライターで月30万円稼げるようになりました。やったことは3つだけ：①高単価案件だけ狙う ②Claudeで効率化 ③クライアントと長期契約　稼げるライターになる方法、固定ツイートで解説してます。

【型③：ストーリー型】

1年前：会社員、副業禁止、貯金50万　転機：こっそりライターを始めた　今：副業月収50万、来月から独立　会社に依存しない生き方、始めてみませんか？ 始め方はプロフから👇

【型④：リスト型】

ライターが稼げない5つの理由：①低単価案件ばかり受けてる ②提案文がテンプレ ③納品して終わり ④専門性がない ⑤営業してない　1つでも当てはまったら、改善方法をプロフで解説してます。

2-4. ClaudeでTwitter投稿を量産する

【プロンプト例：Twitter投稿作成】

あなたはSNSマーケティングのプロです。以下のテーマで、Twitter投稿を10パターン作成してください。

【テーマ】副業ライターで稼ぐ方法 【ターゲット】会社員、副業を始めたい人 【トーン】フレンドリー、共感的

以下の型で、各2パターンずつ作成：①問題提起→解決策型 ②実績・ノウハウ型 ③ストーリー型 ④リスト型 ⑤自由形式

各投稿は140文字以内（日本語全角）で。絵文字は控えめに使用。

第3章：Instagramへの横展開

3-1. Instagramの特徴

フィード投稿（画像/動画＋キャプション）、ストーリーズ、リール、カルーセルの4フォーマット。キャプションは最大2,200文字ですが、表示されるのは最初の3行。画像で止まる→キャプションを読む→保存、シェアが重要な指標。

3-2. Instagram投稿の型

【型①：カルーセル投稿（情報提供型）】

1枚目：タイトル（目を引く）/ 2〜9枚目：内容（1枚1メッセージ）/ 10枚目：CTA。キャプション：1行目に結論orフック、本文：詳細説明、最後：CTA、ハッシュタグ

【型②：ビフォーアフター型】

1枚目：Before→Afterを示す / 2枚目以降：具体的な変化、方法。ダイエット、美容、スキル系で特に効果的。

【型③：共感→解決型】

1枚目：共感を呼ぶ悩み / 2〜3枚目：悩みの深掘り / 4〜8枚目：解決策 / 9〜10枚目：CTA

3-3. Instagramキャプションの書き方

【キャプションの基本構造】

冒頭3行（最重要）：結論、フック、問題提起。「続きを読む」を押させる。本文：詳細説明、改行を多用して読みやすく。CTA：保存してね / フォローしてね / コメントで教えて。ハッシュタグ：関連性の高いもの10〜15個

3-4. ClaudeでInstagram投稿を作成する

【プロンプト例：カルーセル投稿】

あなたはInstagramマーケティングのプロです。以下のテーマで、カルーセル投稿（10枚構成）の原稿を作成してください。

【テーマ】副業ライターが最初の1万円を稼ぐ方法 【ターゲット】副業を始めたい会社員 【トーン】親しみやすい、具体的

・1枚目：タイトル（目を引く、20文字以内）・2〜9枚目：内容（各スライド30〜50文字、1メッセージ）・10枚目：CTA ・キャプション：冒頭3行＋本文＋CTA＋ハッシュタグ15個

第4章：YouTube台本への横展開

4-1. YouTubeの特徴

動画（長尺：5〜20分が主流）＋ショート（60秒以内）。サムネ→タイトル→クリック→冒頭15秒で判断。視聴維持率が最重要。クリック率（CTR）も重要。

4-2. YouTube台本の基本構造

【台本の基本構造】

【オープニング（冒頭15秒〜30秒）】フック：「この動画を見ると○○がわかります」 / 自己紹介（短く）/ 動画の概要

【本編】1つのテーマを深掘り / 章立てで構成 / 各章に「フック」を入れて離脱を防ぐ

【エンディング】まとめ / CTA（チャンネル登録、関連動画）/ 次回予告

4-3. 視聴維持率を高める技術

【冒頭で離脱させない技術】

1. 結論ファースト：最初に「この動画を見るとこうなれる」を伝える

2. ベネフィットを明確に：「3つのポイントを解説します」

3. 自己紹介は短く：5〜10秒で十分

4. ループを作る：先の内容を予告して、見続けさせる

【本編で飽きさせない技術】

5. 章立てを明確に：「1つ目のポイントは…」「2つ目は…」

6. 定期的にフックを入れる：「ここからが本題です」「ここ、超重要です」

7. 具体例を多用：「例えば…」「実際に私は…」

8. 視覚的変化：テロップ、画像、B-roll

4-4. ClaudeでYouTube台本を作成する

【プロンプト例：YouTube台本】

あなたはYouTube動画のプロ構成作家です。以下のテーマで、10分動画の台本を作成してください。

【テーマ】副業ライターが月10万円稼ぐためのロードマップ 【ターゲット】副業を始めたい会社員、20〜40代 【チャンネルのトーン】わかりやすい、親しみやすい

【台本の構成】・オープニング（30秒）・本編（8分、3パートに分けて）・エンディング（1分30秒）

・タイムスタンプ付き ・実際に話す言葉をそのまま書く（口語体）・演出指示も入れる

【プロンプト例：サムネイル・タイトル】

以下の動画のサムネイル用テキストとタイトルを、それぞれ5パターンずつ考えてください。

【動画内容】副業ライターが月10万円稼ぐためのロードマップ 【ターゲット】副業を始めたい会社員

【サムネイルテキスト】15文字以内、数字を入れる、インパクトのある言葉

【タイトル】40文字以内、検索されそうなキーワードを含む、クリックしたくなる要素

4-5. 重要：動画台本スキルの位置づけ

【この章で学べるのは「ライティング」のみです】

この章で学べること：台本の「書き方」（ライティングスキル）/ 視聴者を惹きつける文章構成 / Claudeを活用した効率的な台本執筆。これらは「ライティング案件」としてクライアントから依頼される台本作成に対応するためのスキルです。

【ライティングだけでは動画は伸びません】

台本がどれだけ良くても、それだけでYouTubeチャンネルは伸びません。サムネイルの作り込み、タイトルのキーワード設計、投稿タイミングの最適化、アルゴリズムへの対応、編集技術など、YouTube特有の専門ノウハウが別途必要です。

【YouTubeで本格的に稼ぎたい方へ】

AI副業スクールでは、YouTubeに特化した専門スクールを別途ご用意しています。興味のある方は、LINEにてお気軽にご相談ください。

第5章：TikTok・Reels台本への横展開

5-1. ショート動画の特徴

尺：TikTok 15秒〜3分 / Reels 15秒〜90秒 / YouTubeショート 60秒以内。冒頭1〜2秒で「見るか見ないか」を判断。成功の指標：視聴完了率、リピート再生、エンゲージメント。

5-2. ショート動画の型

【型①：フック→ネタバレ→詳細】

「副業で月30万稼いでます。やってることは、たった1つ。それは…○○です。具体的に説明すると…」

ポイント：結論を先に言って興味を引く

【型②：Before/After型】

「1年前：副業収入0円　今：副業で月30万　変わったきっかけは…」

【型③：○○選・ランキング型】

ランキングは「最後まで見たくなる」心理を利用。1位を最後に持ってくる。

【型④：ストーリーテリング型】

「会社辞めたいって上司に言ったら…実は上司も副業してて…今では一緒にビジネスやってます」

5-3. 冒頭2秒のフック

【数字系】「月30万稼ぐ副業、知りたくないですか？」「99%の人が知らない○○」

【否定系】「副業で稼ぎたいなら、絶対○○するな」「この方法、誰も教えてくれなかった」

【呼びかけ系】「副業始めたい人、ちょっと来て」「会社辞めたい人、聞いて」

【ストーリー系】「年収300万の会社員だった私が…」「副業禁止の会社で、こっそり始めたら…」

5-4. Claudeでショート動画台本を作成する

【プロンプト例：TikTok台本】

あなたはTikTokのバズ動画クリエイターです。以下のテーマで、30秒のショート動画台本を5パターン作成してください。

【テーマ】副業ライターの始め方 【ターゲット】20〜30代、副業に興味がある会社員

・冒頭2秒で強力なフック ・結論ファースト ・30秒で完結 ・口語体

各パターンごとに：使用する型 / 秒数ごとの台本 / 画面構成のイメージ

5-5. 重要：ショート動画台本スキルの位置づけ

この章で学べること：ショート動画台本の「書き方」/ 冒頭で離脱させないフックの作り方 / Claudeを活用した台本の量産方法

台本がどれだけ良くても、それだけでTikTokでバズることはありません。トレンド音源の選び方、TikTokアルゴリズムへの対応、ハッシュタグ戦略、編集技術など、TikTok特有の専門ノウハウが別途必要です。

AI副業スクールでは、TikTokに特化した専門スクールを別途ご用意しています。興味のある方は、LINEにてお気軽にご相談ください。

第6章：Web広告文への横展開

6-1. Web広告の種類と特徴

【リスティング広告】Google、Yahoo!の検索結果に表示。見出し30文字×3＋説明文90文字×2。

【ディスプレイ広告】Webサイトやアプリに表示。ユーザーは「見たくて見ているわけではない」。

【SNS広告】Facebook、Instagram、Twitter、TikTokなど。フィードに溶け込む形式。

【YouTube広告】インストリーム（動画前後、途中）。5秒でスキップ可能なものが多い。

6-2. リスティング広告の書き方

構成：見出し1（30文字、最重要）/ 見出し2（30文字、補足・ベネフィット）/ 見出し3（30文字、CTA・差別化）/ 説明文1・2（各90文字）

ポイント：検索キーワードを含める / 数字を使う / ベネフィットを明確に / CTAを入れる

【リスティング広告の例】

見出し1：副業ライターで月10万円｜未経験OK

見出し2：在宅で始める文章の仕事

見出し3：今すぐ無料で始める方法

説明文1：会社員でも在宅でできる副業ライター。未経験から始めて月10万円稼ぐ人も。まずは無料の入門講座から。

説明文2：受講者1,200名の実績。満足度96%。今なら無料体験実施中。お申し込みはこちらから。

6-3. SNS広告の書き方

【SNS広告テキスト例】

【良い例】「会社員を続けながら、月10万円の副収入を作る方法があります。私も1年前は普通の会社員でした。でも今は、副業で月30万円稼いでいます。始め方を無料で教えます👇」→ ストーリー調で、広告感が薄い

第7章：メルマガ・LINE配信への横展開

7-1. メルマガの特徴と構成

特徴：プッシュ型（届ける）/ 開封率が勝負（件名が最重要）/ 長文でも読まれる / セールスとの相性が良い

基本構成：① 件名（最重要）② 冒頭の挨拶 ③ 本題への導入 ④ 本文 ⑤ CTA ⑥ 署名

【件名の例】

  • 「【残り3名】副業ライター無料相談」
  • 「副業で月10万円稼ぐ人の共通点」
  • 「なぜ、あの人は稼げてあなたは稼げないのか」

7-2. LINE配信の特徴と構成

特徴：開封率が非常に高い（60〜80%）/ 短文が基本（1メッセージ200〜300文字）/ 会話調、親しみやすいトーン

構成：3通セット（1通目：導入 / 2通目：本題 / 3通目：CTA）または1通完結（200〜300文字）

7-3. Claudeでメルマガ・LINEを作成する

【プロンプト例：メルマガ】

あなたはメールマーケティングのプロです。以下のテーマでメルマガを作成してください。

【テーマ】副業ライター講座の募集開始 【ターゲット】副業に興味のあるメルマガ読者 【目的】講座の申し込みを促す

・件名（3パターン、各30文字以内）・本文（800〜1000文字）：冒頭の挨拶、問題提起・共感、講座の紹介、ベネフィット、CTA、署名

【トーン】親しみやすく、でも信頼感がある / 押し売り感を出さない

【プロンプト例：LINE配信】

以下のテーマで、LINE公式アカウントの配信文を作成してください。

3通セットで作成：1通目（100文字）導入 / 2通目（150文字）概要 / 3通目（100文字）CTA

【トーン】友達に話すような口調 / 絵文字は適度に

第8章：LP→SNS→広告の一貫設計

8-1. なぜ一貫設計が重要なのか

  • ① メッセージが統一される：どの媒体で接触しても、同じ印象
  • ② 効率的に制作できる：LP用に作ったコンテンツをSNSに転用
  • ③ CVR（コンバージョン率）が上がる：SNS→LP→購入の流れがスムーズ
  • ④ クライアントへの提案価値が上がる：「LPだけ」より「LP＋SNS＋広告」→単価UP

8-2. 一貫設計のフレームワーク

【一貫設計の5ステップ】

9. コアメッセージを決める：この商品の「一言で言うと何か」

10. ターゲットを明確にする：全媒体共通のペルソナ

11. LP（母艦）を作る：すべての道はLPに通ず

12. LPから各媒体に切り出す：LPの一部をSNS用に短縮

13. 導線を設計する：SNS→LP→購入 の各タッチポイントの役割を明確に

【横展開の心構え】

セールスライティングの原則は、どの媒体でも同じです。「誰に」「何を」「どう伝えるか」この基本さえ押さえれば、どんな媒体にも対応できます。各媒体の「特性」を理解し、「型」を使いこなすこと。

型を知っていれば、Claudeを使って効率的に量産できます。

次章「3-4. 文章×画像×動画をつなぐ設計思考」では、文章だけでなく、画像や動画との連携を考える「設計思考」を学びます。

LPが書けて、SNSが書けて、広告が書けて、動画台本が書ける。あなたは今、「書ける人」から「設計できる人」へと進化しています。この武器を使って、さらに収益を拡大していきましょう。

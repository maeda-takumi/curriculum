フェーズ3｜ライティングを『武器』にする応用編

はじめに：「得意な仕事だけ」では生き残れない時代

フェーズ3もいよいよ後半に入りました。ここまでで、あなたはセールスライティング、LP設計、SNS・動画台本・広告文への横展開、そしてそれらを統合する設計思考と、ライターとしての市場価値を大きく高めるスキルを身につけてきました。

あなたは今、「同じタイプのクライアント」から「同じタイプの案件」ばかり受けていませんか？得意分野があるのは強みです。しかし、1つの分野だけに依存し続けることには、大きなリスクが潜んでいます。

2024年、Googleのコアアップデートにより、SEO記事を主力にしていたライターの仕事が激減しました。

同年、生成AIの普及により「とりあえず文章を書くだけ」の仕事の単価が大幅に下落しました。さらに、特定のクライアントに売上の80%を依存していたライターが、そのクライアントの事業方針の転換で一気に収入を失う事例も後を絶ちません。

【この章の目標】

  • 自分のクライアントワークの「現在の幅」を正確に把握する

  • 既存クライアントから追加案件を1つ以上引き出す
  • 月収の「1社依存率」を50%以下に分散する

第1章：なぜ「幅を広げる」ことが生命線になるのか

1-1. 「1つの柱」に依存するリスク

【リスク①：プラットフォームリスク】

AさんはSEOライターとして月収30万円を安定的に受注していた。しかし2024年3月、GoogleのコアアップデートでクライアントのサイトPVが激減。記事の発注量が月15万文字→3万文字に急落し、月収も30万→6万円に。SEOという1つのプラットフォームへの依存が原因。LP、広告文、SNSなど複数メディアに対応できていれば他でカバーできた。

【リスク②：クライアント集中リスク】

Bさんはメインクライアント1社（全体の80%）に依存していた。担当者退職→内製化方針に変更。月20万円の収入がゼロに。「最大クライアントへの依存率は50%以下」を目指すべき。

【リスク③：スキル陳腐化リスク】

Cさんは5年間、美容系ブログ記事を書き続けてきた。しかし生成AIの登場で「とりあえず情報をまとめるだけの記事」の需要が激減。クライアントが「生成AIで十分」と判断し始めた。LP、セールスレター、広告文、SNS運用のテキスト設計ができなかったCさんは、

「生成AIと同じ程度のことしかできないなら、今の単価は維持できない」と言われてしまった。スキルの幅が狭いと、技術の進化に淘汰される。

1-3. 幅を広げるための3つの心理的ハードルと乗り越え方

【ハードル①：「その業種の知識がない」】

乗り越え方：知識は後からつければいい。クライアントが求めているのは「業界の専門家」ではなく、「自社の商品・サービスの魅力を、顧客に伝わる言葉にしてくれる人」。必要なのは「顧客が何に悩み、何を求めているか」を短時間でリサーチし、言語化する力。

さらに、Claudeを使えば業界リサーチの時間を大幅に短縮できます。（具体的な方法は第8章で解説）

【ハードル②：「新しいメディアの書き方がわからない」】

乗り越え方：メディアが変わっても、ライティングの本質は変わらない。変わるのは「長さ」「トーン」「構成」だけ。新しいメディアに挑戦するときは、まず「そのメディアの成功事例を10個研究する」こと。

【ハードル③：「失敗したらどうしよう」】

乗り越え方：「小さく試す」という戦略。まずは既存クライアントに「ついでにこれも作りましょうか？」と提案する。または、クラウドソーシングで小さな案件を受けて練習する。

第2章：クライアントワークの「幅」を体系的に理解する

2-1. 幅を構成する「3つの軸」

【軸①：業種（誰に書くか）】

美容→感情訴求型 / IT/SaaS→論理訴求型 / 不動産→信頼訴求型 / 教育→共感訴求型 / 飲食→五感訴求型 / 士業→専門性訴求型 / EC→比較訴求型。業種を広げるほど、汎用性の高いライターになれます。

【軸②：メディア（何に書くか）】

SEO記事→検索意図・構成力 / LP→セールスライティング / メルマガ→開封率・関係構築 / SNS投稿→短文の切れ味 / YouTube台本→視聴者維持の構成 / 広告文→限られた文字数での最大訴求 / ホワイトペーパー→B2Bの意思決定支援 / 採用コンテンツ→企業文化の言語化。メディアの幅を広げるほど、1社に対して複数の納品物を提供できるようになります。

【軸③：関わり方（どう関わるか）】

レベル1：単発執筆（作業者）→数千〜数万円 / レベル2：継続執筆（レギュラーライター）→月5〜15万円 / レベル3：設計＋執筆（設計者）→月15〜30万円 / レベル4：コンテンツディレクター→月20〜50万円 / レベル5：マーケティング顧問→月30〜100万円

2-2. 自分の「現在地」を把握する

【自己診断チェックリスト】

【業種の幅チェック】今まで対応した業種をすべて書き出す→（　）業種 / 3業種以下なら「要拡大」

【メディアの幅チェック】今まで書いたことがあるメディア→（　）種類 / 3種類以下なら「要拡大」

【関わり方の深さチェック】現在の主な関わり方は？（レベル1〜5）/ レベル2以下、月額固定契約0なら「要進化」

【収入ポートフォリオチェック】月収の何%が最大クライアントに依存→（　）% / 1社依存率50%超なら「要分散」

2-3. 幅を広げる「戦略マトリクス」

【アンゾフのマトリクス応用版（おすすめの順番）】

1. まず「深掘り」：既存業種×既存メディアで単価を上げる（リスク最小）

2. 次に「メディア拡張」：既存業種×新メディアで同じクライアントからの受注を増やす

3. 次に「業種拡張」：新業種×既存メディアで新しいクライアントを開拓する

4. 最後に「多角化」：新業種×新メディアで完全に新しい領域に挑戦する

焦って④から始めず、①→②→③→④の順番で着実に幅を広げていくのが最もリスクが低く、効果的です。

第3章：業種別「刺さるライティング」の攻略法

⚠ 注意：ここで紹介するのは「各業種のクライアントが求めるライティングの特性」です。専門知識は案件受注後にリサーチで補完しますが、「どんなライティングが求められるか」の理解は事前に必要です。

3-1. 美容・健康業界

特性：購買決定は「感情」が主導 / ビフォーアフターの変化が最大の訴求ポイント / 薬機法による表現規制あり

求められるライティング：お客様の「悩み」に深く共感する文章 / 「変化のストーリー」を描くライティング / 薬機法に抵触しない表現力（「効果がある」→「実感いただいています」）

【文章例】

NG：「この美容液は高濃度ビタミンC配合で、シミを消す効果があります」→ 薬機法に抵触する可能性あり

OK：「鏡を見るたびにため息をついていた40代女性が、3ヶ月後には『ファンデーションの量が減った』と笑顔で話してくれました。まずは7日間お試しください」→ 体験談ベース、断定を避け感情に訴求

3-2. IT・SaaS業界

特性：購買決定は「論理」とROIが主導 / 意思決定者が複数いる / 専門用語が多いが読者全員が専門家ではない

求められるライティング：複雑な機能を「ベネフィット」に翻訳する力 / 数値データを効果的に使った説得力のある文章

【文章例】

NG：「このCRMツールは顧客管理ができます。機能はA、B、Cがあります」→ 機能の羅列

OK：「営業チーム30人が、毎月20時間かけていた顧客情報の手入力。導入から2ヶ月でその作業時間がゼロになりました。浮いた600時間/月を商談に充てた結果、成約率が1.4倍に改善」→ 具体的な数字・ビフォーアフター・ベネフィットが明確

3-3. 不動産業界

特性：人生最大の買い物→信頼感・安心感が最重要 / 「比較・検討」期間が長い / 宅建業法による広告表示の規制あり

求められるライティング：「安心感」「信頼感」を文章で演出する力 / 物件の特徴を「暮らしのイメージ」に変換する力

【文章例】

NG：「駅から徒歩10分。3LDK、築15年。日当たり良好。近くにスーパーあり」→ スペックの羅列

OK：「朝、南向きのリビングに差し込む陽光で目を覚ます。子どもたちの声が聞こえる公園まで徒歩3分、帰り道にある24時間スーパーで食材を買えば家族の笑顔が待つ夕食の時間。駅徒歩10分の利便性を兼ね備えた3LDKです」→ 暮らしのストーリーを描きスペックも伝達

3-4. 教育・スクール業界

特性：「変化（ビフォーアフター）」が最大の訴求力 / 受講者の声・成功事例が決定打 / 不安の解消（自分にもできるか？）が購買の壁

求められるライティング：受講生の「変化の物語」を描くストーリーライティング / 「あなたにもできる」という確信を生む文章

3-5. EC・物販業界

特性：「比較」が前提（Amazon、楽天で他の商品と必ず比べられる）/ 購入の決め手は口コミ・写真・価格・説明文の4要素

求められるライティング：「スペック」を「ベネフィット」に翻訳する力 / 購入の背中を押す「限定感」「緊急性」の演出

3-6. 士業（税理士・弁護士・社労士等）

特性：「信頼性」「専門性」が最も重要 / 見込み客は「問題を抱えた状態」で検索してくる / 敷居の高さを下げる必要がある

求められるライティング：専門用語を噛み砕いて説明する「翻訳力」 / 「この先生なら安心」と思わせる人柄の言語化

3-7. 飲食・店舗ビジネス

特性：来店動機は「写真」「口コミ」「立地」の3要素 / 「五感」で訴求（味、香り、食感、雰囲気）/ リピート率が経営の生命線

求められるライティング：五感に訴える描写力（「シズル感」のある文章）/ SNSに映える短文コピー

3-9. 新しい業種で「最初の一本」を書くときの実践テクニック

【テクニック①：その業種の「口コミ」を100件読む】

Googleマップ・X・Yahoo!知恵袋・Amazon/楽天レビューを最低100件読む。100件読むと「この業界の顧客が使う言葉」が自分の中にインストールされる。そこで拾った「リアルな言葉」をそのままキャッチコピーに使うと、業界経験10年のライターにも負けない文章が書ける。

【テクニック②：競合LPを「構造分解」する】

競合LPをセクション番号・目的・見出しコピー・文字数・画像内容・感情の変化に分解してメモする。これを5社分やると「この業界のLPの勝ちパターン」が浮かび上がる。

【テクニック③：クライアントの「既存顧客」に仮想インタビューする】

クライアントが持っているアンケートデータをもらう / Googleマップや口コミを全部読む / クライアントの営業担当に「お客様がよく言う言葉は？」と聞く / Claudeに「このペルソナなら何と言うか」をシミュレーションしてもらう

【テクニック④：「3つのBefore/After」を準備する】

①数字のB/A「月間問い合わせ5件→25件に」②感情のB/A「毎日不安で眠れなかった→安心して過ごせている」③行動のB/A「何から始めればいいかわからなかった→毎日30分の習慣ができた」。この3つを準備しておけばどのメディアにも応用できる。

【テクニック⑤：「専門用語辞書」を自作する】

用語・正式な意味・エンドユーザーにわかりやすい言い換え・使用上の注意点を辞書形式でまとめておく。クライアントごとに蓄積すると次の同業種案件に即対応できる。

第4章：「1クライアントから複数案件を引き出す」技術

クライアントワークの幅を広げる最も効率的な方法は「既存クライアントからの受注を増やす」ことです。マーケティングの世界では「新規顧客の獲得コストは既存顧客の維持コストの5倍」と言われます。

4-1. 既存クライアントの「潜在ニーズ」を発見する方法

【潜在ニーズ発見の5つの視点】

【視点①：クライアントの「導線」を辿る】クライアントのWebサイト・SNS・広告を「自分がこの会社の見込み客だったら」という目線で体験する。「導線の穴」を見つけたら、そこが提案のチャンス。

【視点②：「やりたいけど手が回っていない」ことを聞く】「他にやりたいけど手が回っていないことはありますか？」この質問だけで追加案件が生まれることが多い。（SNS開始・メルマガ・YouTube台本等）

【視点③：納品物の「周辺」に目を向ける】LP原稿を納品したとき、LPに誘導する広告文 / LPから登録した人へのメルマガ / LP用のSNS告知投稿 / 効果測定後の改善案が「周辺」に存在する。

【視点④：クライアントの「競合」を観察する】競合がやっていてクライアントがやっていないことは、そのままクライアントへの提案ネタになる。

【視点⑤：季節・イベントに合わせた提案】年末年始キャンペーン用LP・メール / 新年度の採用コンテンツ / ボーナス時期のセールコピー。時期に合わせた「先回り提案」はクライアントに感動される。

4-2. アップセル・クロスセルの具体的な提案フレーズ

【場面別・提案フレーズ集】

LP原稿を納品するとき：「LP原稿、お納めします。ちなみに、このLPに誘導するためのSNS告知投稿のテキストもご用意しましょうか？LPとトーンを合わせて書いた方が、クリック率が上がりやすいので」

SEO記事を継続で書いているとき：「アクセスの多い記事をベースにしたメルマガを作ると、既存顧客へのリピート促進に効果がありそうです。メルマガの文面作成も私の方でできますが、いかがですか？」

月末確認報告メール：「来月は御社の○○キャンペーンの時期ですよね。キャンペーン用のLP・メール・SNS投稿をまとめて設計・執筆するプランもご用意できます。来週あたり30分お時間いただけませんか？」

4-3. 「提案→受注」の成功確率を高める5つの原則

【原則①：まず「成果」で信頼を積む】追加提案が通るかどうかは現在の仕事での信頼次第。まずは目の前の案件で期待を超える品質を出すことが最優先。

【原則②：「無料のおまけ」から始める】まずは「おまけ」として無料で価値を提供する。例：LP原稿に画像指示書を無料で添付する。クライアントが「おっ、これいいな」と思えば次から「正式に依頼したい」となる。

【原則③：提案は「1つだけ」に絞る】「あれもできます、これもできます」は逆効果。最も効果が高い1つに絞る。

【原則④：クライアントの「痛み」に紐づける】×「SNS投稿の代行もできます」（自分目線）→ ○「SNSからLPへの流入が少ないようなので、投稿テキストの設計でお力になれます」（クライアント目線）

【原則⑤：数字で効果を示す】「同業他社の事例では、メルマガ導入後にリピート率が15%向上した例があります」。数字があるとクライアントは社内で稟議を通しやすい。

第5章：新しい業種・メディアに挑戦する実践手順

5-1. 新業種に挑戦する「5ステップ」

【STEP1：業界リサーチ（3〜5時間）】

業界基本構造の理解 / 業界用語20〜30個の把握 / 主要プレイヤー10社のWebサイト・LP・SNSを確認 / エンドユーザーの口コミ30件を読む

Claudeプロンプト例：

「[業種名]業界について教えてください。以下の項目でまとめてください：1.業界の基本構造 2.エンドユーザーが最も多い悩み・課題トップ10 3.業界特有の専門用語30個とその意味 4.主要企業10社とその特徴 5.この業界で効果的なマーケティング手法 6.この業界でライターが注意すべき法規制や慣習」

【STEP2：競合LP・広告の研究（2〜3時間）】

競合他社のLPを5本以上精読。広告ライブラリ（Meta Ad Library、Google広告透明性センター）でその業界の広告文を20本以上チェック。「効いていそうな表現」と「弱い表現」を仕分ける。

【STEP3：サンプル制作（3〜5時間）】

架空のクライアントを設定し、LP原稿を1本・SNS投稿案を5本書いてみる。ポートフォリオに「○○業界対応」として掲載。クライアントへの提案時に「こんな文章が書けます」と見せられる。

【STEP4：小さな案件で実戦（実案件）】

クラウドソーシングでその業種の小規模案件を受注する。最初は単価が低くても「経験値を買う」つもりで受ける。応募文にSTEP3で作ったサンプルを添付する。「未経験です」とは書かず「当業界のリサーチを行い、サンプルを制作しました」と書く。

【STEP5：ポートフォリオに追加＆本格展開】

実績をポートフォリオに追加。「○○業界のLP原稿実績あり」と掲載するだけで、同業界からの問い合わせが増える。1つの業種あたり2〜4週間で回せば、3ヶ月で2〜3業種を追加できます。

第6章：「関わり方」を進化させるキャリアステップ

6-1. 5段階のキャリアステップ

【レベル1：単発作業者（月収3〜8万円）】依頼された原稿を書いて納品する。クライアントとの関係は「一期一会」が多い。

【レベル2：継続ライター（月収8〜15万円）】毎月一定量の原稿を担当。月額固定契約。クライアントの事業をある程度理解している。

【レベル3：設計者ライター（月収15〜30万円）】何を書くか、どう書くかの「設計」から担当。画像指示書やSNSテキスト設計まで含めた統合提案。

【レベル4：コンテンツディレクター（月収20〜50万円）】他のライターの原稿も管理・品質チェックする。「自分が書く」から「人に書かせる」へのマインドシフト。

【レベル5：マーケティング顧問（月収30〜100万円）】コンテンツ戦略全体のアドバイザーとして関わる。月1〜2回のミーティングで戦略の方向性をアドバイス。

6-2. 各レベルの「昇格テスト」

【レベル1→2への昇格テスト（3つ以上当てはまれば準備OK）】

  • □ 同じクライアントから3回以上リピート発注された
  • □ クライアントが自分の名前を覚えてくれている
  • □ 納品後に「いつもありがとう」的なメッセージが来る
  • □ 自分から「次はこんなのも書きましょうか」と提案したことがある

【レベル2→3への昇格テスト（4つ以上当てはまれば準備OK）】

  • □ クライアントの事業の「売上目標」や「KPI」を把握している
  • □ クライアントから「○○さんの意見を聞きたい」と言われたことがある
  • □ 納品物に「画像指示書」や「SNSテキスト案」をおまけで添付したことがある
  • □ 自分の書いた文章の「成果」（PV、CVR等）を確認したことがある

6-3. 「レベル3（設計者）」を最速で目指すアクションプラン

【3ヶ月でレベル3に到達するロードマップ】

1ヶ月目：既存クライアントの事業を徹底リサーチ / すべての納品に「おまけ」を付ける / 月末に「改善提案メール」を1通送る

2ヶ月目：既存クライアントに設計込みの追加案件を1つ提案する / 新しい業種またはメディアの案件を1つ受注する / Claudeを活用した設計業務のワークフローを確立する

3ヶ月目：設計込みの案件を正式に受注し成果を出す / ポートフォリオを「設計者ライター」仕様にリニューアルする / 月額固定契約を1つ以上新規獲得する

7-2. ポートフォリオを育てる戦略

フェーズ1：種まき（月収5〜10万円）クライアント2〜3社を確保。「経験値を積む」ことを優先。

フェーズ2：深掘り（月収10〜20万円）最も相性の良いクライアントに「追加提案」を開始。1社からの月額を5→10万円に引き上げる。

フェーズ4：最適化（月収30万円〜）低単価案件を手放し高単価案件に集中。レベル3以上の案件比率を50%以上に。Claudeを活用して作業時間を短縮し、時間単価を上げる。

8-2. Claudeとの「壁打ち」で品質を上げる

Claudeは単なる「文章生成ツール」ではなく、ライターの「壁打ちパートナー」として活用すると真価を発揮します。

【壁打ちプロンプトの例】

以下のLP原稿のファーストビューについて、プロのコピーライターの視点でフィードバックをください。（ここに自分が書いた文章を入力）。以下の観点で：1.ターゲットの「痛み」に十分刺さっているか 2.最初の一文で「読み進めたい」と思えるか 3.ベネフィットが具体的に伝わっているか 4.不要な言葉・冗長な表現はないか 5.CTAの文言は行動を促すのに十分か 6.改善案を3つ提案してください。

Claudeからのフィードバックを鵜呑みにする必要はありません。しかし「別の視点」を素早く得られる点で、品質向上の強力なパートナーになります。

特に未経験の業種に挑戦するときは、Claudeに「この業界のプロとして」フィードバックをもらうことで、業界感覚を素早くキャッチアップできます。

3-2. LP・セールス文章をClaudeで設計する

フェーズ3｜ライティングを『武器』にする応用編

はじめに：LPは「設計図」で決まる

前章では、セールスライティングの「基本構造」を学びました。この章では、その理論を「実践」に落とし込みます。

具体的には、「LP（ランディングページ）」と「セールス文章」を、Claudeを活用しながら設計・作成する方法を学びます。

【LPとは何か】

LP（Landing Page：ランディングページ）とは、広告やリンクをクリックした人が「最初に着地するページ」のことです。一般的には、1つの商品・サービスを売るために作られた「縦長の1ページ完結型のWebページ」を指します。LPの目的は「訪問者に行動を起こさせること」です。

【なぜLPを学ぶのか】

  • ① 高単価案件の代表格：LP制作は、1本5万円〜50万円という高単価案件が多い
  • ② あらゆるセールス文章の基礎：LPが書ければ、広告文、メール、SNSなど、あらゆるセールス文章に応用できる
  • ③ Claudeとの相性が良い：構造が明確なため、Claudeを活用しやすい
  • ④ 成果が数字で見える：コンバージョン率という形で、成果が明確にわかる

【この章の目標】

  • LPの基本構造を完全に理解する
  • Claudeを使ってLP原稿を効率的に作成できる
  • クライアントに提案できるレベルのLP設計ができる
  • 実際にLP原稿を1本完成させる

第1章：LPの基本構造

LPには、長年の研究と実践で確立された「黄金構造」があります。

1-1. LPの全体構成

【LPの基本パーツ】

1. ファーストビュー（FV）→ 最初に目に入る部分。キャッチコピー、メイン画像

2. 問題提起・共感 → 読者の悩みに共感し、問題を明確化

3. 解決策の提示 → 商品・サービスを解決策として提示

4. ベネフィット → 得られる未来、メリットを伝える

5. 商品・サービス説明 → 具体的な内容、特徴を説明

6. 実績・証拠 → お客様の声、データ、メディア掲載など

7. 料金・オファー → 価格、特典、保証を提示

8. よくある質問（FAQ）→ 購入前の不安を解消

9. クロージング・CTA → 行動を促す最後の一押し

1-2. 各パーツの役割と目的

【① ファーストビュー（FV）】

役割：読者の注意を引く / 「自分に関係がある」と思わせる / 続きを読みたいと思わせる

含まれる要素：キャッチコピー（最重要）/ サブキャッチ / メイン画像 / CTAボタン（任意）

【② 問題提起・共感】

役割：読者の悩みを言語化する / 「この人は私のことをわかっている」と思わせる / 問題の深刻さを認識させる

【③ 解決策の提示】

役割：悩みを解決する方法があることを伝える / 商品・サービスを「解決策」として位置づける / 期待感を高める

【④ ベネフィット】

役割：商品を使った後の「理想の未来」を見せる / 「欲しい」という感情を高める / 行動する理由を与える

【⑤ 商品・サービス説明】

役割：具体的な内容を伝える / 「何が含まれているか」を明確にする / 論理的な判断材料を提供する

【⑥ 実績・証拠】

役割：信頼性を高める / 「本当に効果がある」と納得させる / NOT BELIEVEの壁を超える

【⑦ 料金・オファー】

役割：価格を提示する / お得感を感じさせる / 今買う理由を作る

【⑧ よくある質問（FAQ）】

役割：購入前の不安、疑問を解消する / 最後の障壁を取り除く / 信頼感を高める

【⑨ クロージング・CTA】

役割：最後の一押しをする / 具体的な行動を促す / NOT ACTの壁を超える

第2章：LP設計の5ステップ

LPを作成する前に、「設計」が必要です。設計なしに書き始めると、まとまりのない文章になってしまいます。

【LP設計の5ステップ】

STEP 1：ターゲットを明確にする / STEP 2：商品・サービスを整理する / STEP 3：ベネフィットを洗い出す / STEP 4：構成（ワイヤーフレーム）を作る / STEP 5：各パーツの原稿を作成する

2-1. STEP1：ターゲットを明確にする

ターゲット設定で決めること：① 基本属性（年齢、性別、職業、年収）② 悩み・課題（今の悩み・感情・過去に試した解決策）③ 願望・理想（なりたい自分・実現したい生活）④ 購買状況（商品を知ったきっかけ・躊躇する理由）

2-2. STEP2：商品・サービスを整理する

商品情報の整理項目：① 基本情報（商品名・価格・提供形態）② 内容・特徴（含まれるもの・他社との違い・独自の強み）③ 実績（販売数・満足度・メディア掲載）④ 保証・サポート（返金保証・アフターサポート）

2-3. STEP3：ベネフィットを洗い出す

商品の「特徴（フィーチャー）」を「ベネフィット」に変換します。「だから何？」と自問を繰り返すことで、読者にとっての価値を見つけます。

2-4. STEP4：構成（ワイヤーフレーム）を作る

ターゲット・商品情報・ベネフィットが整理できたら、LPの構成を決めます。9つのパーツ（FV→問題提起→解決策→ベネフィット→商品説明→実績→料金→FAQ→クロージング）の順で組み立てます。

2-5. STEP5：各パーツの原稿を作成する

構成が決まったら、各パーツの原稿を作成します。ここからが、Claudeの出番です。

第3章：Claudeでファーストビューを作る

ファーストビュー（FV）は、LPの「顔」です。ここで読者の心をつかめなければ、その先は読まれません。

3-1. キャッチコピーの重要性

キャッチコピーの目的は、ただ1つ。「次の1行を読ませること」。3秒で「読むか読まないか」が決まります。

3-2. Claudeでキャッチコピーを量産する

Claudeを使えば、キャッチコピーのアイデアを大量に出すことができます。

【プロンプト例①：基本形】

あなたはプロのコピーライターです。以下の商品のキャッチコピーを20個考えてください。

【商品】ビジネス英語オンライン講座 【ターゲット】35歳男性、海外案件担当になったが英語に自信がない 【ベネフィット】3ヶ月で海外クライアントと自信を持って会話できる 【価格】98,000円

以下の切り口で、各4個ずつ作成してください：① ターゲットの悩みに呼びかける ② ベネフィットを直接伝える ③ 好奇心を刺激する ④ 数字を使う ⑤ 恐怖・損失回避に訴える

【プロンプト例②：型を指定】

以下の「型」を使って、キャッチコピーを作成してください。

型①「○○で悩んでいませんか？」/ 型②「○○する方法」/ 型③「なぜ○○は△△なのか」/ 型④「○○だけが知っている」/ 型⑤「たった○○で△△できる」/ 型⑥「○○しなくても△△できる」/ 型⑦「○○年間で○○人が実証した」

各型で2パターンずつ、合計14個作成してください。

【プロンプト例③：既存コピーの改善】

以下のキャッチコピーを、より効果的なものに改善してください。

【現在のコピー】「ビジネス英語が身につくオンライン講座」

【改善の方向性】・より具体的に ・ベネフィットを明確に ・ターゲットの悩みに寄り添う

5パターンの改善案を出してください。それぞれ、なぜその改善が効果的か説明も付けてください。

3-3. キャッチコピーの選び方

【良いキャッチコピーの5つの条件】

10. ターゲットが「自分のことだ」と感じる → 悩み、願望が具体的に表現されている

11. ベネフィットが明確 → 読むと何が得られるかわかる

12. 具体的で信憑性がある → 数字、期間など具体性がある

13. 好奇心を刺激する → 「どういうこと？」と思わせる

14. 短くてわかりやすい → 一瞬で理解できる

3-4. サブキャッチとその他の要素

【プロンプト例：サブキャッチ作成】

以下のキャッチコピーに合うサブキャッチを5個考えてください。

キャッチコピー：「海外会議で、堂々と発言できる自分へ」

サブキャッチには以下を含めてください：・具体的な期間や数字 ・信頼性を高める要素 ・ターゲットを明確にする言葉

第4章：Claudeで問題提起・共感パートを作る

4-1. 問題提起・共感パートの役割

このパートの役割は、読者に「この人は私のことをわかっている」と思わせることです。

15. 読者の悩みを言語化する → 「そうそう、まさにそれ！」と思わせる

16. 問題の深刻さを認識させる → 「このままじゃまずい」と感じさせる

17. 共感・信頼を築く → 「この人はわかってくれる」と思わせる

18. 解決策への期待を高める → 「どうすればいいの？」と思わせる

4-2. Claudeで悩みリストを作る

【プロンプト例：悩みリスト作成】

あなたはプロのコピーライターです。以下のターゲットが抱えている「悩み」を30個リストアップしてください。

【ターゲット】・35歳男性、IT企業の営業課長 ・海外案件の担当になったが、英語に自信がない ・過去に英会話スクールに通ったが、続かなかった

以下のカテゴリに分けて出力してください：① 仕事での具体的な困りごと（10個）② それによる感情・心理（10個）③ 将来への不安（10個）

各悩みは、ターゲットが「そうそう！」と共感する具体的な表現で書いてください。

4-3. 共感パートの文章を作る

【プロンプト例：共感パート作成】

以下の悩みリストを元に、LPの「問題提起・共感」パートを作成してください。

【悩みリスト】・海外クライアントとのオンライン会議が憂鬱でたまらない ・言いたいことがあるのに、英語で出てこない ・上司から「英語力上げろ」とプレッシャーをかけられる ・会議中、発言できない自分が恥ずかしい ・後輩に追い抜かれるかもしれない焦り

【書き方の指示】・「こんなお悩みありませんか？」から始める ・悩みは箇条書きで5〜7個 ・その後、共感のメッセージを2〜3文 ・「実は、私も同じでした」という共感ストーリーを100文字程度 ・最後に「でも、大丈夫です」という希望のメッセージ

4-4. 問題を深掘りする

【プロンプト例：問題の深掘り】

以下の悩みを放置すると、どんな「最悪の未来」が待っているか、5つのシナリオを具体的に描写してください。

【悩み】海外案件担当なのに、英語に自信がない

・具体的で生々しい描写 ・ただし、過度に恐怖を煽らない ・最後は「でも、そうならないための方法があります」で締める

第5章：Claudeで解決策・ベネフィットパートを作る

5-1. 解決策パートの作り方

【プロンプト例：解決策パート】

以下の情報を元に、LPの「解決策の提示」パートを作成してください。

【商品】ビジネス英語マスター講座 【ターゲットの悩み】海外会議で英語が話せない

【書き方の指示】・「その悩み、○○で解決できます」から始める ・なぜ解決できるのか、3つの理由を説明 ・各理由は見出し＋3〜4文の説明 ・読者が「これなら解決できそう」と期待を持てる内容

5-2. ベネフィットパートの作り方

【プロンプト例：ベネフィットパート】

以下の商品のベネフィットを、LPの「ベネフィット」パートとして作成してください。

【商品】ビジネス英語マスター講座 【ターゲット】35歳男性、海外案件担当、英語に自信がない

【書き方の指示】・「受講後、あなたはこう変わります」という見出し ・各ベネフィットを「Before → After」形式で表現 ・具体的な場面を描写 ・感情的な変化も含める

5-3. Before/Afterストーリーを作る

【プロンプト例：Before/Afterストーリー】

以下のターゲットの「Before/Afterストーリー」を作成してください。

【ターゲット】・35歳男性、IT企業の営業課長 ・海外案件担当になったが、英語に自信がない

【書き方の指示】・Beforeパート（200文字）：悩んでいる時の具体的な場面 ・転機パート（50文字）：講座に出会ったきっかけ ・Afterパート（200文字）：3ヶ月後の変化

第6章：Claudeで商品説明・実績パートを作る

6-1. 商品説明パートの作り方

【プロンプト例：商品説明パート】

以下の商品情報を元に、LPの「商品説明」パートを作成してください。

【商品情報】・商品名：ビジネス英語マスター講座 ・期間：3ヶ月 ・内容：動画講座36本（1本15分）/ 週1回オンライングループレッスン（60分）/ 専用チャットで質問し放題 / オリジナルテキスト（PDF）・講師：外資系企業で10年勤務した現役ビジネスパーソン

【書き方の指示】・「講座内容」という見出し ・各コンテンツをわかりやすく説明 ・ベネフィットと紐づけて説明する

6-2. 実績・証拠パートの作り方

【プロンプト例：お客様の声】

以下の情報を元に、LPに掲載する「お客様の声」を3名分作成してください。

【商品】ビジネス英語マスター講座 【ターゲット】35歳男性、海外案件担当、英語に自信がない

【書き方の指示】・各150〜200文字 ・Before→Afterの変化を含める ・具体的なエピソードを入れる ・3名は異なる属性（年齢、業界、悩み）にする ・名前、年齢、職業を設定（架空でOK）

※注意：これは例として作成するものです。実際のLPでは、本物のお客様の声を使用してください。

6-3. データ・実績の見せ方

【プロンプト例：実績データの文章化】

以下の実績データを、LPで効果的に見せる文章に変換してください。

【実績データ】・累計受講者数：1,200名 ・受講者満足度：96% ・TOEIC平均スコアアップ：150点 ・受講継続率：94% ・日経新聞掲載

【書き方の指示】・数字を目立たせる ・信頼性を高める補足を入れる ・ターゲットが「自分もこうなれる」と思える表現

第7章：Claudeで料金・オファーパートを作る

7-1. 料金提示の心理学

① 価格の妥当性を説明する ② 比較で安く見せる ③ 分割で心理的負担を下げる ④ 特典で価値を積み上げる ⑤ 保証でリスクを取り除く

7-2. Claudeで料金パートを作る

【プロンプト例：料金パート】

以下の情報を元に、LPの「料金・オファー」パートを作成してください。

【価格情報】・通常価格：128,000円 ・今回限定価格：98,000円 ・分割払い：月々8,500円×12回

【特典】・特典①：ビジネス英語フレーズ集（PDF）・特典②：発音矯正動画講座 ・特典③：修了後の1ヶ月延長サポート

【保証】・30日間全額返金保証

【書き方の指示】・価格の妥当性を説明（英会話スクールとの比較など）・特典の価値を金額で表現 ・分割払いで心理的負担を下げる表現 ・保証で安心感を与える ・「今すぐ」申し込む理由を作る

7-3. 限定性・緊急性の作り方

【プロンプト例：限定性・緊急性】

以下の商品の「限定性・緊急性」を訴求する文章を作成してください。

【商品】ビジネス英語マスター講座 【価格】98,000円（期間限定）

【限定要素】・先着30名様限定 ・今月末までの申し込み限定 ・次回募集は3ヶ月後

【書き方の指示】・なぜ限定なのか、理由を説明（説得力を持たせる）・煽りすぎず、事実として伝える ・ただし、嘘の限定性は絶対にNG

第8章：ClaudeでFAQ・クロージングを作る

8-1. FAQの作り方

【プロンプト例：FAQ作成】

以下の商品について、LPに掲載する「よくある質問（FAQ）」を10個作成してください。

【商品】ビジネス英語マスター講座 【ターゲット】35歳男性、海外案件担当、英語に自信がない 【価格】98,000円

【書き方の指示】・購入を検討している人が抱く疑問 ・ネガティブな質問にも正直に答える ・回答で不安を解消し、安心感を与える

8-2. クロージングの作り方

【プロンプト例：クロージング】

以下の情報を元に、LPの「クロージング」パートを作成してください。

【商品】ビジネス英語マスター講座 【ターゲット】35歳男性、海外案件担当、英語に自信がない 【ベネフィット】3ヶ月で海外クライアントと自信を持って会話できる

【書き方の指示】・最後の背中を押すメッセージ ・ベネフィットを再度強調 ・行動しないリスクも伝える ・明確なCTA（申し込みボタンの文言）・追伸（P.S.）で重要ポイントを念押し

8-3. CTAボタンの文言

【ベネフィット訴求型】

  • 「3ヶ月後の自分を変える」
  • 「英語力を手に入れる」
  • 「海外会議で活躍する自分になる」

【安心感訴求型】

  • 「30日間リスクなしで試す」
  • 「まずは無料で相談する」

【緊急性訴求型】

  • 「期間限定価格で申し込む」
  • 「残り○席、今すぐ確保する」

第9章：LP全体をClaudeで作成する

ここまで、各パーツの作り方を学びました。この章では、LP全体を一気に作成するプロンプトを紹介します。

9-1. LP全体作成プロンプト

【マスタープロンプト】

あなたはプロのセールスコピーライターです。以下の情報を元に、LP（ランディングページ）の原稿を作成してください。

【商品情報】・商品名：[商品名を入力] ・価格：[価格を入力] ・内容：[内容を入力] ・特徴：[特徴を入力] ・実績：[実績を入力]

【ターゲット情報】・ターゲット：[ターゲットを入力] ・悩み：[悩みを入力] ・願望：[願望を入力]

【LP構成】以下の構成で作成してください：

1. ファーストビュー：キャッチコピー（20文字以内）/ サブキャッチ（40文字以内）

2. 問題提起・共感（300文字）：「こんなお悩みありませんか？」/ 悩みリスト5〜7個 / 共感メッセージ

【LP作成の心構え】

LPは「書く」のではなく「設計する」ものです。設計 ＝ ターゲット × 商品 × 構造。この3つが揃えば、Claudeを活用して効率的に高品質なLPを作成できます。ただし、Claudeの出力をそのまま使わないこと。必ず人間の目でチェックし、修正することが重要です。

第2章：DALL-E（OpenAI）で画像生成を始める〔35分〕

2-1. DALL-E 3を使うための準備

✅ DALL-E 3を使う条件

【必要なもの】OpenAI ChatGPT Plus（月額$20 ≒ 約3,000円）または ChatGPT Team/Enterprise

【無料版では使えない？】ChatGPT無料版ではDALL-E 3は使用不可。代替：Bing Image Creator（無料）でDALL-E 3が使える → https://www.bing.com/create → Microsoftアカウントでログインして使用

2-2. OpenAI ChatGPTでDALL-E 3を使う手順

Step1：ChatGPTにログイン → https://chat.openai.com にアクセス → ChatGPT Plusアカウントでログイン

Step2：新しいチャットを開始 → 左上の「New chat」をクリック

Step3：画像生成を依頼 → 日本語で「〇〇の画像を作って」と入力。例：「海辺でコーヒーを飲む女性の写真を作って」

Step4：画像が生成される → 数秒〜十数秒で画像が表示される

Step5：ダウンロード → 画像をクリック→ダウンロードボタン

2-3. 最初の画像を生成してみよう

🎨 最初に試すプロンプト例

【例1】シンプルな依頼：「かわいい猫のイラストを作って」

【例2】少し具体的な依頼：「カフェでノートパソコンを使っている女性の写真風画像を作って」

【例3】スタイルを指定：「水彩画風の富士山と桜の風景を作って」

2-4. 生成された画像の調整

方法1：追加指示を出す（「背景をもっと明るくして」「人物を左側に配置して」「もう少しポップな雰囲気にして」）

方法2：バリエーションを依頼（「同じテーマで違うパターンを3つ作って」「色違いのバージョンを作って」）

方法3：部分修正を依頼（「空の色だけ夕焼け色に変えて」「テキストを削除して」）

※会話形式なので、自然な日本語で指示できます

2-5. Bing Image Creator（無料代替）の使い方

Step1：Bing Image Creatorにアクセス → https://www.bing.com/create

Step2：Microsoftアカウントでログイン（アカウントがなければ無料で作成）

Step3：プロンプトを入力 → 英語推奨（日本語も可）→「Create」ボタンをクリック

Step4：4枚の画像が生成される → 気に入った画像をクリックしてダウンロード

注意点：1日の生成回数に制限あり / 商用利用の可否はMicrosoftの規約を確認

3-5. 避けるべきNGプロンプト

NG1：有名人・著名人の名前を使う（肖像権、パブリシティ権の侵害になる可能性）→ 代わりに特徴を説明

NG2：著作権のあるキャラクター（「ピカチュウ」「ミッキーマウス」等）→ オリジナルキャラクターとして説明

NG3：暴力的・性的な内容 → AIが生成を拒否、またはアカウント停止のリスク

NG4：曖昧すぎるプロンプト（「いい感じの画像」）→ 具体的に説明することで精度UP

第8章：著作権・商用利用の注意点〔20分〕

8-1. AI生成画像の著作権

日本の現状：AIが自動生成した画像には著作権が発生しない可能性が高い。ただし、プロンプトの工夫や編集を加えた場合は創作性が認められる可能性あり。法律の解釈はまだ確定していない。

実務上の注意：他人のAI生成画像を無断で使わない / 自分で生成した画像は基本的に自由に使える / ただし各ツールの利用規約を必ず確認

NG3：他人が生成した画像を自分の作品として販売

NG4：実在の商品・ブランドのロゴを生成して使用（商標権侵害になる可能性）

NG5：違法・有害なコンテンツの生成（アカウント停止、法的責任の可能性）

8-4. 安全に使うためのチェックリスト

  • □ 使用するツールの利用規約を読んだ
  • □ 商用利用が許可されていることを確認した
  • □ 有名人・著名人の名前は使っていない
  • □ 既存のキャラクター・ブランドを模倣していない
  • □ 人物の顔がある場合、架空の人物であることを明記
  • □ 販売する場合、AI生成であることを必要に応じて開示

第9章：実践ワーク：オリジナル素材を10点作成する〔40分〕

9-1. ワーク概要

このワークのゴール：画像生成AIの操作に慣れる / プロンプトの書き方を実践で身につける / すぐに使えるオリジナル素材を10点作る / Canvaとの組み合わせを体験する

【作成する10点の内訳】背景素材：3点 / 商品イメージ：3点 / SNS用イラスト：2点 / アイキャッチ用：2点

9-2. 事前準備

  • □ DALL-E 3（ChatGPT Plus）または Bing Image Creator にログインできる
  • □ Canvaにログインできる
  • □ 作りたい画像のイメージを決めた

9-3. ワーク①：背景素材を3点作成

【プロンプト例1：グラデーション背景】「パステルピンクからパステルブルーへのグラデーション背景。柔らかい雰囲気、ミニマル、SNS投稿用」

【プロンプト例2：テクスチャ背景】「大理石風のテクスチャ背景。白とグレー、高級感、商品撮影の背景として使える」

【プロンプト例3：自然背景】「ボケた緑の葉っぱの背景。自然光、爽やかな雰囲気、被写体が映えるシンプルな構図」

● □ 3点とも生成できたらダウンロードして保存

9-4. ワーク②：商品イメージ素材を3点作成

【プロンプト例1：コスメ】「白い大理石の上に置かれたリップスティック。ピンクの花びらが散らばっている。商品撮影風、高級感」

【プロンプト例2：ファッション】「おしゃれなカフェでコーヒーを持つ女性の手元。アクセサリーが見える。写真風、暖かい雰囲気」

【プロンプト例3：インテリア】「北欧風のシンプルな部屋。白い棚に観葉植物と本。自然光、ミニマル、Instagram風」

● □ 3点とも生成できたらダウンロードして保存

9-5. ワーク③：SNS用イラストを2点作成

【例1：アイコン的イラスト】「コーヒーカップのシンプルなイラスト。フラットデザイン、パステルカラー、白背景」

【例2：キャラクター的イラスト】「本を読んでいるかわいい猫のイラスト。柔らかいタッチ、温かい色使い、白背景」

  • □ 2点とも生成できたらダウンロードして保存
  • □ 背景が白の場合、remove.bgで背景を透過処理

9-6. ワーク④：アイキャッチ用素材を2点作成

【例1：ビジネス系】「ノートパソコンとスマートフォンがあるデスク。コーヒーとメモ帳。上から見た構図。明るい自然光、すっきりとした雰囲気」

【例2：ライフスタイル系】「窓辺で本を読む人のシルエット。夕日の光、温かい雰囲気、リラックス」

● □ 2点とも生成できたらダウンロードして保存

9-7. ワーク⑤：Canvaで仕上げる

  • □ 生成した10点をCanvaにアップロード
  • □ 背景素材を使って、テキスト入りの画像を1点作成（キャッチコピーを配置）
  • □ 商品イメージ素材に、価格や商品名を追加して1点作成
  • □ イラスト素材を使って、SNS投稿用画像を1点作成（背景＋イラスト＋テキストの組み合わせ）
  • □ 3点以上の完成画像をダウンロード
  • □ 楽天ROOMまたはSNSに1点以上投稿

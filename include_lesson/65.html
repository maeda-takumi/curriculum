テスト課題：副業アイデアリストを作成しよう

1. 課題の目的

  • 副業に関するアイデアを一元管理できる仕組みを作る
  • ChatGPTを活用して「発想の幅」を広げる
  • Notionでデータベース化し、評価・優先度をつけて整理できるようにする

2. 提出内容

提出物

副業アイデアリストを作成したNotionのスクリーンショット

3. ChatGPTへの依頼例（プロンプト例）

発散型（幅広くアイデアを出す）

副業として取り組めるアイデアを30個出してください。

カテゴリ（スキル系／販売系／投資系／情報発信系）ごとに整理してください。

個人特化型（自分の経験を活かす）

私は接客業の経験が5年あります。

その経験を活かした副業アイデアを10個提案してください。

出てきたアイデアはそのままNotionへ登録すればOK。

4. Notionでの作成手順

Step1. ページを作成

  • 新規ページ → タイトル「副業アイデアリスト」
  • アイコンに設定を入れると分かりやすい

Step2. データベース（テーブル）を作る

プロパティ例

  • アイデア名（Title）
  • カテゴリ（ブログ／物販／スキル販売／動画編集 etc.）
  • 初期コスト（低／中／高）
  • 実現可能性（1〜5）
  • メモ

これで副業アイデアの一覧表が完成。

Step3. 評価と整理

  • 「実現可能性」で並べ替え
  • 「短期で出来るもの」「中長期で挑戦するもの」に分ける
  • 必要に応じて「優先度」「リスク」「工数」などの列を追加してもOK

Step4. 定期的にアップデート

  • 毎週／毎月 ChatGPTに相談して新しいアイデアを追加
  • 古いアイデアの「今の現実性」を見直す
  • 実行したものは「メモ」欄に進捗を記録

5. 実践例（サンプル）

アイデア名 カテゴリ 初期コスト 実現可能性 メモ
ブログ運営 情報発信 低 ★★★★☆ ChatGPTで構成作成可
ハンドメイド販売 物販 中 ★★★☆☆ Canvaで画像作成
動画編集代行 スキル 中 ★★★★★ 需要が高い
SNS運用代行 情報発信 低 ★★★★☆ Notionに投稿管理表を作る

これをNotionに入れて、Canvaでアイデアカード化すると副業ポートフォリオが完成。

6. まとめ
  • Notion：アイデアを整理・評価していつでも見返せる資産化
  • ChatGPT：自分では思いつかない方向のアイデアが増える

副業の第一歩として最適なワークフローになります。

7. PCでのスクリーンショットの注意点

Windows

  1. Windowsキー + Shift + S を押す
  2. 画面が暗くなる
  3. マウスで撮りたい範囲をドラッグ
  4. 自動でクリップボードに保存
  5. Ctrl + V で貼り付け

Mac

  1. Command + Shift + 4
  2. カーソルが十字になる
  3. ドラッグで範囲選択
  4. デスクトップに保存

8. スクリーンショットが撮影できない方へ（Notionリンク共有方法）

作ったばかりのページは他の方に見えない状態なので、以下の操作で共有できる状態に変更してください。

  1. PCの場合は右上の「共有」ボタン

    スマホの場合は右上の「シェア」ボタンを押す

  2. 「招待されたユーザーのみ」と書かれた部分を押す

  3. 「Web上でリンクを知る全ユーザー」と書かれた部分を押す

  4. PCの場合下記の画像のようになっていればOK

    「リンクをコピー」を押して、そのまま送りたい相手にリンクを共有して下さい。

    スマホの場合は「リンクを共有」というボタンから送りたい相手に共有して下さい。

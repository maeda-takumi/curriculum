実践課題1 「ChatGPTに原稿を書かせ、CapCutで編集」

1. 課題の目的

  • ChatGPTで動画用の原稿（スクリプト）を作成する
  • 原稿を元に、自分の声やナレーションで動画化する
  • CapCutでテロップ・BGM・カット編集を加えて仕上げる
  • 「AIで原稿をつくり、編集で完成させる」流れを体験する

提出は不要。完成品は自分用の練習・SNS投稿の素材として活用できます。

2. やり方の流れ

Step1. ChatGPTで原稿を作る

  • ChatGPTに「60秒動画用の台本を書いて」と依頼する
  • プロンプト例

    テーマは「朝の集中力を高める方法」。

    TikTok用に60秒の台本を作って。

    冒頭で惹きつける一言 → ポイント3つ → 最後にまとめ。

    テロップに使えるように短文でお願いします。

出力された原稿を少し調整して、自分の言葉に近づけてもOK。

Step2. 撮影または音声収録

  • 原稿を読み上げて、自分の声で録画する（スマホでOK）
  • 顔出しが苦手な場合は「声＋背景動画」でも良い
  • 噛んでも大丈夫。CapCutで後からカットできる

Step3. CapCutで編集

  1. 新規プロジェクトを作成し、動画や音声を取り込む
  2. 不要部分をカット（噛んだ所や無音部分を削除）
  3. テロップ挿入
    • ChatGPTの原稿をコピペして字幕にする
    • 1行12〜18文字で改行すると読みやすい
    • 白文字＋黒縁や半透明帯で視認性を上げる
  4. BGM追加
    • CapCut内の音楽から選択
    • 声が主役なのでBGM音量は40〜60%に調整
  5. 効果音や簡単なズームを入れてリズムを出す

Step4. 書き出しと保存

  • 書き出し形式：MP4
  • 縦型 1080×1920（TikTok/リール用）がおすすめ
  • 完成した動画はカメラロールに保存して、SNSに投稿またはNotionにストック

3. 実践例（イメージ）

ChatGPT原稿（抜粋）

  • 「朝に集中できない？ 実はカンタンに変えられます！」
  • 「1. 朝日を浴びる」
  • 「2. 水を一杯飲む」
  • 「3. 5分だけ運動する」
  • 「この3つで、朝から最高の1日が始まります！」

CapCut編集の流れ

  • 冒頭にタイトルカード「朝の集中法3選」
  • 自分の声＋テロップ（白文字・黒縁）
  • 軽快なBGMを下に流す
  • 最後に締めカード「今日からやってみよう！」

これだけで「60秒のショート動画」が完成します。

4. ポイントとコツ

  • ChatGPTに「短文で出力して」と伝えるとテロップに使いやすい
  • CapCutの自動字幕機能を使ってもOK（あとで誤字修正）
  • Canvaで冒頭・締めカードを作ると完成度がさらに高まる
この課題は提出不要です。目的は「AIを原稿係にして、自分は編集者になる」感覚を掴むことです。

数本試すと「撮影前に台本があるととても楽」という実感が持てるようになります。

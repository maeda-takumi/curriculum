テスト課題：自分の副業アイデアをファネルに落とし込む

1. 課題の目的

  • 自分の副業アイデアを「顧客の流れ（ファネル）」で整理する
  • 「知る → 興味 → 検討 → 購入 → リピート → 紹介」の流れを自分で設計できるようになる
  • ChatGPTで整理 → Notionで構造化 → Canvaで図解 → 提出という一連の流れを体験する

2. 提出内容

提出物：

  1. ファネル表を作成したNotionの公開ページのリンク
  2. Canvaで作成したファネル図（PDF書き出し）

3. やり方ステップ

Step1. 副業アイデアを1つ選ぶ

  • 例：物販／ブログ／ハンドメイド／動画編集代行／コーチングサービス

Step2. ChatGPTでファネルを整理

プロンプト例：

私の副業アイデアは「ハンドメイドアクセサリー販売」です。

このビジネスをファネルに落とし込み、

「知る → 興味 → 検討 → 購入 → リピート → 紹介」の流れを整理してください。

ChatGPTが各ステップの行動や施策を提案してくれる

Step3. Notionで表にまとめる

提出用には表にすると見やすい。

ファネル段階 顧客の行動例 自分の施策／工夫
認知（知る） Instagramで広告を見つける SNS投稿・Canvaで作った画像広告
興味（関心） 投稿に「いいね」や保存をする ChatGPTで作った商品説明キャプション
検討 他のブランドと比較する Notionで商品一覧＆FAQをまとめる
購入 ECサイトから購入する BASEやSTORESで販売
リピート 気に入って再度購入 定期便・クーポン発行
紹介（拡散） 友人にシェアする レビュー投稿で特典

Step4. Canvaで図解

  • Canvaで「ピラミッド型」「矢印型」のファネル図を選ぶ
  • 「知る → 興味 → 検討 → 購入 → リピート → 紹介」を順番に配置
  • 自分の副業アイデアを各段階に書き込む

図解すると、提出資料としても見やすくなる

4. 実例（3つ）

1) ハンドメイドアクセサリー販売

  • 認知：Instagramで作品を見つける
  • 興味：投稿保存／プロフィール閲覧
  • 検討：他店と比較 → Notionで商品一覧を提示
  • 購入：STORESで購入
  • リピート：季節限定品をリリース
  • 紹介：友人に口コミ、レビュー特典

2) ブログ×アフィリエイト

  • 認知：Google検索で記事を発見
  • 興味：記事を読む／SNSでフォロー
  • 検討：比較記事・ランキングを見る
  • 購入：アフィリエイトリンク経由で購入
  • リピート：別ジャンル記事を継続的に読む
  • 紹介：SNSで記事シェア

3) 動画編集代行サービス

  • 認知：TwitterやTikTokで実績動画を発信
  • 興味：DMやコメントで相談
  • 検討：価格表やサンプルを見比べる
  • 購入：単発で編集依頼
  • リピート：定期契約（月額プラン）
  • 紹介：クライアントが別の人に紹介

まとめ

  • ファネルは「お客様の旅路」を見える化する仕組み
  • ChatGPT → Notion → Canva の順で使うと整理しやすい
  • 提出時は「表＋図＋補足メモ」の3点セットがベスト

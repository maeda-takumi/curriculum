Week1 ②ChatGPTの基本

プロンプトとは何か？

プロンプトの意味

ChatGPTに何かを頼むときに使う「指示文」のことをプロンプトといいます。たとえば、「ChatGPTとは何ですか？」という質問もプロンプトの一種です。また、「小学生にもわかるように100文字で説明して」というように条件をつけると、より自分の求める答えに近づけます。

プロンプトとは、ChatGPTに「どんな答えを出してほしいか」を伝えるための“会話の言葉”です。プロンプトが上手になると、AIが出す答えの質が大きく変わります。副業で記事作成やマーケティングを行うときも、プロンプトが的確であれば、時間をかけずに高品質な成果を作れます。

なぜプロンプトが大事なのか

ChatGPTは魔法のように答えを出してくれるツールですが、実際には「人間の指示に忠実なアシスタント」です。「曖昧な質問」をすると、返ってくる答えも曖昧になります。逆に、「具体的で明確な指示」を出すと、驚くほど正確で使える答えを返してくれます。

たとえば、「歴史を教えて」では広すぎて答えがぼんやりします。でも「高校生にも分かるように、江戸時代の特徴を200文字で説明して」と言えば、目的がはっきり伝わり、使いやすい文章が返ってきます。副業でライティングや記事作成をする場合、プロンプトの質が「成果物の質」を決めます。プロンプトが上手な人ほど、仕事のスピードと正確さが上がり、報酬アップにつながります。

初心者のよくある失敗

ChatGPTを使い始めたばかりの人によくあるのが、次の3つの失敗です。

  1. ざっくりしすぎる質問
    例：「歴史を教えて」→ ChatGPTはどの時代、どの国、どの視点か分からず、広すぎる内容を返してしまいます。
  2. 条件不足
    例：「文章を書いて」→ どんな長さで、誰向けに、どんな文体で書くのかが不明確だと、結果も使いにくくなります。
  3. 一回で完璧を求める
    ChatGPTは「一度に完璧な回答を出す」よりも、「会話を重ねて調整する」ことで理想の答えを導くのが基本です。

この3つを避けるだけでも、ChatGPTの活用力がぐんと上がります。

プロンプトの基本構造

良いプロンプトには「3つの要素」があります。

  1. 目的（何をしたいのか）
  2. 対象（誰に向けた内容か）
  3. 条件（形式・長さ・文体など）

たとえば、「高校生向けに、江戸時代の歴史を200字でまとめて、箇条書きで出して」というプロンプトは、目的・対象・条件がすべて含まれています。このように指示を整理して伝えるだけで、ChatGPTの回答が劇的にわかりやすくなります。副業でも、クライアントから求められる「トーン・形式・ボリューム」を反映しやすくなり、修正の手間が減ります。

プロンプトを“レシピ”に例える

ChatGPTを料理人、あなたをお客さんと考えてみましょう。あなたが出す「注文書」がプロンプトです。「カレー作って」だけでは、辛さや量が分かりません。でも「中辛・2人前・じゃがいも多め・20分以内で」と伝えれば、あなた好みの料理が出てきます。

同じように、「ビジネス向けに、丁寧なトーンで、500文字以内のメールを作って」と指示すれば、目的にぴったりの文が生成されます。つまり、プロンプトは“AIに正しく仕事をしてもらうためのレシピ”なのです。

よく使う基本プロンプトの型と例

説明系

ChatGPTに何かを説明してもらうときに使う型です。副業で「解説記事」や「教育コンテンツ」を作る際に非常に役立ちます。

例：
・AIとは何か、小学生でも分かるように100字で説明して。
・マーケティングを3つの例え話で説明して。

こうした説明型プロンプトは、初心者でもすぐに使える基本形です。

要約系

長い文章を短く整理してもらうプロンプトです。ブログ記事、報告書、会議メモなどをまとめるときに便利です。

例：
・以下の文章を300字で要約して。
・ポイントだけ箇条書きで抽出して。

文章作成の副業では、納品前にChatGPTに要約させて「重要ポイントを抜けなくチェックする」使い方も効果的です。

比較系

二つのものの違いを分かりやすくまとめるときに使います。記事や資料作成、副業講座の教材づくりにも向いています。

例：
・株式取引とFXの違いを初心者向けに説明して。
・インスタグラムとX（旧Twitter）の特徴を表にまとめて。

比較系は、初心者が情報を整理するのに最適なプロンプトです。

文章生成系

ChatGPTの得意分野です。メール、ブログ、セールス文、自己紹介文など、幅広く使えます。

例：
・上司に残業のお願いをする丁寧なメールを作成して。
・恋愛に悩む人向けに、共感を込めたブログ記事を1000字で。

副業ライターやSNS運用代行では、この型が最もよく使われます。

変換系

既存の文章を別の形に直すときに使います。文体変更、翻訳、敬語化、カジュアル化などが可能です。

例：
・この文章を「ですます調」に直してください。
・以下の文を英語に翻訳して、カジュアルにしてください。

クライアントの要望に合わせてトーンを変える副業では、非常に便利な機能です。

発想支援系

新しいアイデアを出してもらうためのプロンプトです。商品名、タイトル、キャッチコピーなどを考える副業に活かせます。

例：
・新しいYouTubeチャンネルの名前案を10個出して。
・カフェのキャッチコピーを考えて。

発想系のプロンプトは、アイデア出しに時間がかかる人の強い味方です。

会話例

ユーザー：歴史を分かりやすく教えて。

ChatGPT：どの時代が知りたいですか？

ユーザー：江戸時代の文化を小学生向けに説明して。

ChatGPT：江戸時代は歌舞伎や浮世絵が人気だった時代です。今のテレビや漫画のように、人々を楽しませる文化が広がりました。

このように、ChatGPTとの会話は「やり取りを重ねて理想の答えを作る」ことがポイントです。

プロンプトを進化させるコツと未来

レベルアップのコツ

プロンプトを上達させるコツは、次の3つです。

  • 条件を細かくする：「300字で」「初心者向けに」「箇条書きで」など
  • 視点を変える：「先生として説明して」「営業マンの立場で話して」など
  • 段階的に依頼する：「まず概要を説明→次に具体例→最後にまとめ」

これを意識するだけで、ChatGPTが出す文章の精度が一気に上がります。

応用テクニック

  • ロールプレイ：ChatGPTに役割を与える。
    例：「あなたはプロのマーケターです。この商品をどう宣伝しますか？」
  • スタイル指定：文体を変える。
    例：「ラップ調で説明して」「童話風にまとめて」
  • 出力形式：形式を指定する。
    例：「Markdown形式で」「表にまとめて」

副業で資料を作ったり、SNS投稿を整えたりする際に役立ちます。

プロンプト改善の流れ

  1. まずざっくり依頼する
  2. 出力を見て修正指示を出す
  3. 何度か調整して完成形に近づける

ChatGPTは対話型ツールです。一度で完璧を求めず、会話の中で精度を上げることが大切です。

未来のプロンプト

今後のChatGPTは、音声や画像も理解できるようになります。「この写真の内容を説明して」「この図のデータを要約して」など、より直感的なやり取りが可能になります。また、個人ごとの好みや文体を学習し、使うほどあなたに合わせた答え方をするようになります。

会話例（応用）

ユーザー：あなたはプロのコピーライターです。新しい健康ドリンクのキャッチコピーを5個出して。

ChatGPT：① 朝を変える一杯 / ② 疲れ知らずの相棒 / ③ 健康は、飲む習慣から / ④ 元気を、手軽にチャージ / ⑤ 飲むほどに軽やかに。

ユーザー：もっと若者向けに、SNSでバズる感じで。

ChatGPT：① #朝活エナジー / ② #飲むだけ革命 / ③ #ヘルシーにゴー / ④ #毎朝リセット / ⑤ #チャージして今日も最高！

このように、条件を加えて指示することで、出力のテイストを自由に変えられます。

まとめ

プロンプトとは、ChatGPTへの指示文のことです。指示の出し方次第で、AIの出す答えの品質が大きく変わります。初心者は、説明・要約・比較・文章生成・変換・アイデア出しの基本型から始めましょう。慣れてきたら、条件指定・ロールプレイ・出力形式などを組み合わせて、より高度な使い方を身につけましょう。プロンプトは、AI時代の新しいスキルであり、練習するほど副業や仕事で役立つ「言葉の武器」になります。

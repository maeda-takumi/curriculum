Week7 ④CapCut × VOICEVOXで作る ナレーション動画入門

ナレーション動画の基礎を理解する

ナレーション動画とは？

ナレーション動画は「映像＋音声解説」で進むタイプの動画のことです。

  • YouTubeの解説動画
  • Instagram Reelsで説明する動画
  • 学習教材、商品紹介、スライド動画

など、どんなジャンルでも応用できます。

特に「声を録音するのが苦手」という初心者でも、VOICEVOXのAIナレーションを使えば誰でも自然な声の動画が作れます。

【VOICEVOXのダウンロード】

PCの場合
https://voicevox.hiroshiba.jp/

スマホの場合
https://www.voicevox.su-shiki.com/

なぜ CapCut × VOICEVOX が最強なのか

この2つを組み合わせると、初心者でも質の高い作品が作りやすくなります。

VOICEVOX

  • 無料で使える
  • 種類豊富なナレーション
  • 自然で聞きやすい声質

CapCut

  • スマホで直感操作
  • 字幕が自動生成
  • テンプレートが豊富
  • 動画編集の基本が全部できる
動画構成の基本

ナレーション動画は、以下の順番で作ると視聴者が離れにくいです。

  • 導入：結論やメッセージ
  • 本編：情報やストーリー
  • まとめ：次のアクションを伝える（例：続きはプロフィールへ）

スマホだけでも可能

VOICEVOXはパソコンが無くても使用可能です。

スマホに「CapCut」を入れて、ナレーション音声を「VOICEVOX」で作るだけ。

まとめ
  • ナレーション動画は「声 × 映像」の組み合わせ
  • 自分の声が苦手でもAIでナレーションが作れる
  • CapCutとVOICEVOXは初心者と相性抜群
VOICEVOXでナレーションを作成する

VOICEVOX の基本操作

  1. VOICEVOX を開く

  2. 話者（声の種類）を選ぶ

  3. 読み上げてほしい文章を入力

  4. 「再生」で確認

  5. 問題なければ音声を書き出し（WAV形式）

初心者へのおすすめ話者

  • 四国めたん（落ち着いた雰囲気）
  • ずんだもん（明るい、親しみやすい）
  • 春日部つむぎ（ナレーション向き）

読みやすくする工夫

音声にメリハリをつけるために、以下のテクニックを文章に追加します。

  • 「、」で息継ぎポイントをつくる
  • 重要ワードの前にスペースを入れる
  • 言い回しは短くする

例：

「今日のテーマは、動画編集です。初心者でもできる方法を、5分で解説します。」

音声の調整ポイント

  • speed（読み上げ速度）→ 少し早め（1.1〜1.2）がお勧め
  • pitch（声の高さ）→ 程よく聞きやすい高さに
  • intonation（抑揚）→ 0.9〜1.0くらいが自然
まとめ
  • 文章入力 → 声選択 → 調整 → 音声書き出し
  • 癖のない声を使うと動画に合いやすい
  • 文章はシンプルに、短く区切ると聞きやすい
CapCutで映像にナレーションを載せる

ナレーション動画編集の流れ

  1. 新規プロジェクトを作る

  2. VOICEVOXの音声を読み込む

  3. 映像素材（写真・動画）を配置

  4. 音声に合わせてカット編集

  5. 字幕を自動生成し、読みやすく調整

  6. BGMを入れて音量バランスを整える

  7. 書き出して完成

ナレーションをベースに編集する

動画制作では「音声が先」。音声に合わせて映像を乗せるとテンポが崩れません。

※例えば、話し終わったら次の映像が流れる等

  • 音声をタイムラインに配置
  • 波形を見ながら映像を乗せる
  • テンポに合わせて写真を切り替える

字幕の作り方（超重要）

字幕は動画のわかりやすさを決めます。

  1. 自動の場合：CapCut → キャプション → 自動キャプション
    手動の場合：CapCut → テキスト → テキストを追加
  2. 誤字を修正
  3. 行間を調整（短くする）
  4. 色・背景を整える（白背景 × 黒字が読みやすい）

字幕の最適ルール

  • 1行は15文字以内
  • 2行以内
  • 句読点の位置に合わせる

BGMと効果音の入れ方

BGM

  • 音量はナレーションの「10〜15％」
  • 静かめの曲がおすすめ

効果音

  • ページ切り替え
  • 強調ポイント
  • ポップ音

→ 入れすぎると逆に安っぽくなるため「少しだけ」

書き出し設定

SNSごとにおすすめ設定が異なります。

  • YouTube → 1920×1080（横）
  • TikTok / Reels → 1080×1920（縦）
  • 画質 → 1080p・高画質
動画クオリティを一気に高めるコツ

テンポは「少し早い」くらいが丁度良い

視聴者はスキップ癖があるため、テンポの遅い動画はすぐ離脱されます。

情報を詰め込みすぎない

1動画＝1テーマが基本です。

良い例：「ナレーションの作り方」

悪い例：「ナレーション・編集・分析・アップロード全部まとめ」

最初の3秒に命をかける

冒頭で視聴者の興味を奪うと最後まで見てもらえる。

例：

  • 「今日は、あなたの動画が“急に伸びる理由”を解説します。」
  • 「この編集ミス、やってませんか？」

テスト課題：自分の名刺を作ろう！

1. 課題の目的

  • 自分の基本情報（氏名・連絡先・SNSなど）を整理し、わかりやすくまとめる
  • Canvaの名刺テンプレートを使って、シンプルでプロ風のデザインを作成できるようになる
  • 情報整理 → デザイン → 活用までの一連の流れを体験する

2. 提出内容

提出物：

Canvaで作成した名刺デザイン（PNGまたはPDF）

3. ChatGPTへの依頼例（プロンプト例）

入力例

名刺に入れる情報を整理したいです。

以下をベースに、シンプルで見やすい順番に並べてください。

氏名：〇〇

メールアドレス：〇〇

電話番号：〇〇

SNSアカウント（X / Instagram）：〇〇

見やすい順番の提示を受けて、そのままCanvaに反映する。

4. Canvaでの作成手順

  1. Canvaを開く
  2. 検索欄で「名刺」と入力
  3. 好きなテンプレートを選ぶ（シンプル系が使いやすい）
  4. テンプレートの文字部分を自分の情報に置き換える

デザインのコツ

  • 名前は一番大きく
  • メール／電話番号は少し小さく
  • SNSはアイコン使用で視認性UP
  • 余白を十分に取り、詰め込まない
  • 色は2〜3色以内、フォントは読みやすさ優先

5. 出力例（実例サンプル）

例：佐藤 一郎 の名刺

  • 名前：佐藤 一郎
  • メール：satoichiro@example.com
  • 電話番号：090-1234-xxxx
  • SNS：Instagram @sato1ro_life

デザイン例（Canva）

  • 白背景＋青アクセント
  • 左側に名前、右下に連絡先
  • SNSアイコンを使用

シンプルで信頼感のあるデザインに仕上がる。

6. 保存・活用方法

  • PDF：印刷用（プリントショップ・コンビニプリントで利用可能）
  • PNG：SNSプロフィール画像／メール署名に貼り付けて使用可能
まとめ

名刺作成は情報整理 → テンプレ差し替え → デザイン微調整の流れで簡単に完成する

作った名刺は自己紹介カードやポートフォリオにも応用可能

ChatGPTで整理 → Canvaで形にする、という実践的スキルが身につく
